# 美式卡通具体表情写法

用途：为 `组图 23 表情库.md` 的后续重构提供写法草案。核心思路是：顶层类别保持简洁情绪词，如 `生气`、`委屈`、`喜欢`、`开心`；每个类别下方再写 8 个更具体、更好玩的美式卡通脸部变体。

## 简短系统提示词

你服务于卡通休闲找不同游戏。请为给定的顶层表情类别生成美式卡通脸部表情模板。

写法规则：

- 顶层类别用简洁情绪词，不把目录拆成过多细碎状态词；例如用 `生气`，不要把 `气到脸红`、`气到冒烟`、`咬牙爆筋` 全部拆成顶层类别。
- 每个顶层类别固定包含 8 个具体表情变体；8 个变体必须有不同的视觉锚点，避免同义换词。
- 模板行使用 `编号. 具体表情变体；字段：……；字段：……` 格式；`具体表情变体` 必须与上方 8 个变体名一一对应。
- 模板采用主视觉锚点优先写法：每条写 3-5 个最能体现该表情的脸部字段，不固定字段顺序，不强制写满眉 / 眼 / 脸颊 / 嘴 / 脸侧。
- 每条必须有 1 个最醒目的主视觉锚点，字段顺序按主锚点优先；至少包含 `眼：` 或 `嘴：` 之一；每个字段最多出现 1 次。
- 模板字段以脸部锚点为主，可使用 `眉：`、`眼：`、`脸颊：`、`额头：`、`嘴：`、`牙齿：`、`舌头：`、`口边：`、`脸侧：` 等字段。
- 只写脸部卡通效果，不改身体、姿态、头部朝向、四肢动作、头发、衣服、饰品、背景或地面。
- 不写外来覆盖物或附属区内容，例如面具、口罩、眼罩、头套、纸片、布条、项链、领角、脖子饰品、耳饰或头饰；这些属于组图 23 的人物点位规则，不进入普通表情库。
- 正向好看、负向丑萌：`喜欢 / 期待 / 惊喜 / 崇拜 / 感动 / 害羞 / 卖萌` 优先写漂亮、亮眼、脸红、甜笑、小露齿；`害怕 / 社死 / 吃瘪 / 反胃 / 嘴硬失败 / 晕眩 / 僵住` 可写豆豆眼、黑线、冷蓝 / 冷青额头、小鸭嘴、牙齿打颤等丑萌恶搞锚点。
- `偷笑 / 奸笑 / 看戏 / 幸灾乐祸 / 成功蒙混 / 顽皮` 走调皮坏笑层：单边挑眉、斜眼、压不住嘴角、单颗尖牙、舌尖等可以用，但仍只停留在脸部字段。
- 卡通效果只允许停留在脸部或脸侧极近区域；不得写成大面积污渍、真实伤口、血泊、衣物沾染、喷溅到画外或背景。
- 允许使用脸部表情比喻，如 `小鸭嘴形瘪嘴`、`豆豆眼`、`星星瞳孔`、`爱心瞳孔`、`旋涡瞳孔`、`锯齿牙`、`下巴小幅掉开`、`花瓣红晕`；不得写成真实变形、长出外来器官或动物部位。
- 美式卡通感优先使用大眼、露齿、鼓嘴、小鸭嘴瘪嘴、吐舌、流口水、脸红、额头青筋、短烟线、旋涡眼、豆豆眼、下巴小幅掉开、牙齿打颤等可一眼画出的视觉锚点。
- `额头：` 可写青筋、短烟线、黑线、冷灰 / 冷蓝 / 冷青小色块、小汗；必须是额头局部卡通情绪标记，不写整脸变色、真实光影或发光效果。
- `口边：` 可写口水、呕吐短线、吐气线、卡通红色短线；必须停在嘴边，不扩散到衣物、背景或画外。
- 单人模板的眼神优先绑定 `[目标物]` 或 `[证据物]`；多人模板可绑定 `[对方人物]`，也可同时回指 `[目标物]` 或 `[证据物]`。
- 1-2 为中等可见，3-4 为明显，5-6 为强烈，7-8 为夸张但仍合理。
- 每条至少有一处不对称；不要让 8 条只变成“眉眼嘴强度递增”的重复版本。
- 落库前需确认表情抽取工具已支持任一脸部字段开头；若工具尚未适配，临时模板仍需包含 `眉：` 字段，避免解析漏读。

## 主锚点写法示例

以下示例用于说明“字段随表情服务”，不是新增顶层类别。

```text
委屈 - 小鸭嘴瘪住；嘴：嘴巴向前瘪成小鸭嘴形，嘴唇圆圆鼓出；眼：双眼湿漉漉看向[目标物]；眉：双眉向内上拱；脸颊：双颊被瘪嘴挤得微微鼓起。

不服 - 鼓脸鸭嘴；嘴：嘴巴瘪成圆圆小鸭嘴，像把话憋住；脸颊：一侧脸颊鼓成气包；眼：一眼瞪向[证据物]，另一眼半眯；眉：一侧眉头压低。

奸笑 - 单边尖齿坏笑；嘴：嘴角向一边斜斜拉起；牙齿：一侧露出小小尖齿线；眼：一眼半眯看向[目标物]，另一眼坏坏收窄；眉：一侧眉峰高高挑起。

社死 - 黑线空白眼；额头：额头垂下几条短短卡通黑线；眼：双眼空白看向[证据物]；嘴：嘴巴僵成细缝；脸颊：脸颊僵住并失去弧度。

反胃 - 冷青吐舌脸；额头：额头上半部出现一小片冷青色难受阴影；舌头：舌头软软歪出；眼：眼皮发软避开[目标物]；口边：嘴边冒出一小段卡通呕吐短线，只停在嘴边。
```

## 风格分层

- 漂亮正向：适用于 `喜欢`、`期待`、`惊喜`、`崇拜`、`感动`、`害羞`、`卖萌`。优先用爱心瞳孔、星星瞳孔、亮眼、小鹿圆眼、脸红、甜笑、小露齿、花瓣红晕；不要写成惊恐、崩坏、反胃或丑化。
- 调皮坏笑：适用于 `偷笑`、`奸笑`、`忍笑`、`看戏`、`幸灾乐祸`、`成功蒙混`、`顽皮`。优先用单边挑眉、半眯眼、斜眼、歪嘴坏笑、单颗尖牙、舌尖小得意。
- 丑萌恶搞：适用于 `害怕`、`嘴硬失败`、`社死`、`无语`、`僵住`、`晕眩`、`吃瘪`、`委屈`、`反胃`。可用小鸭嘴瘪住、牙齿打颤、冷蓝 / 冷青额头、黑线空白眼、豆豆眼、旋涡眼、嘴边呕吐短线；仍不得写身体动作或外来物。

## 案例一：生气

### 生气

适用场景：被惹到、不服、争执、目标被破坏或对方行为太离谱。

具体表情变体：
1. 气到脸红
2. 咬牙爆筋
3. 气鼓鼓
4. 气到冒烟
5. 怒到露齿
6. 脸红憋气
7. 气到吐血短线
8. 笑不出来的怒脸

**单人模板 1-8（看向证据物 / 目标物）**
1. 气到脸红；眉：一侧眉头压低，另一侧眉尾挑起；眼：一侧眼瞪住[目标物]，另一侧眼收窄；脸颊：两侧脸颊明显涨红；嘴：嘴巴抿紧成下压斜线。
2. 咬牙爆筋；眉：双眉向眉心挤紧，一侧眉峰更高；眼：双眼怒视[证据物]；额头：额角鼓起一小段卡通青筋；牙齿：上下牙咬成锯齿线。
3. 气鼓鼓；眉：一侧眉尾下压，另一侧眉尾绷直；眼：一眼盯住[目标物]，另一眼半眯；脸颊：双颊鼓成气包；嘴：嘴巴闭紧成圆鼓形。
4. 气到冒烟；眉：双眉低压并形成尖锐折角；眼：双眼瞪向[证据物]，眼尾紧收；额头：额头上方冒两段短短卡通烟线；嘴：嘴角向下绷住。
5. 怒到露齿；眉：一侧眉心深深压下，另一侧眉峰上拱；眼：一侧眼锁住[目标物]，另一侧眼发硬；牙齿：一侧露出咬紧的牙齿；嘴：嘴角向一边拉紧。
6. 脸红憋气；眉：眉头向下挤住，眉尾明显发抖；眼：双眼憋红般瞪住[证据物]；脸颊：脸颊鼓起并泛红；嘴：嘴巴闭紧，像把话硬憋回去。
7. 气到吐血短线；眉：双眉大幅下压，一侧眉尾抽起；眼：双眼怒瞪[目标物]，眼神僵住；嘴：嘴巴猛地张开，吐出一小股卡通红色短线；口边：红色短线只停在嘴边，不扩散。
8. 笑不出来的怒脸；眉：一侧眉峰高高挑起，另一侧眉头压得很低；眼：一眼冷冷盯着[证据物]，另一眼半垂；嘴：嘴角想笑却僵住，牙齿露出一小段怒笑齿线。

**多人模板 1-8（看向对方人物）**
1. 气到脸红；眉：一侧眉头压低，另一侧眉尾挑起；眼：一侧眼瞪住[对方人物]，另一侧眼扫向[证据物]；脸颊：两侧脸颊明显涨红；嘴：嘴巴抿紧成下压斜线。

## 案例二：喜欢

### 喜欢

适用场景：心动、宠爱、明显偏爱、被可见人物 / 动物 / 食物 / 礼物 / 目标物打动。

具体表情变体：
1. 爱心眼心动
2. 脸红偷看
3. 亮晶晶圆眼
4. 甜到眯眼
5. 捧场大笑脸
6. 舌尖小期待
7. 口水心动
8. 脸颊开花笑

**单人模板 1-8（看向证据物 / 目标物）**
1. 爱心眼心动；眉：双眉向外舒展开，一侧眉尾更高；眼：双眼变成清楚爱心形看向[目标物]；脸颊：脸颊泛起圆圆红晕；嘴：嘴巴开心张成小弧口。
