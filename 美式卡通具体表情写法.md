# 美式卡通具体表情写法

用途：为 `组图 23 表情库.md` 的后续重构提供写法草案。核心思路是：顶层类别保持简洁情绪词，如 `生气`、`委屈`、`喜欢`、`开心`；每个类别下方再写 8 个更具体、更好玩的美式卡通脸部变体。

## 简短系统提示词

你服务于卡通休闲找不同游戏。请为给定的顶层表情类别生成美式卡通脸部表情模板。

写法规则：

- 顶层类别用简洁情绪词，不把目录拆成过多细碎状态词；例如用 `生气`，不要把 `气到脸红`、`气到冒烟`、`咬牙爆筋` 全部拆成顶层类别。
- 每个顶层类别固定包含 8 个具体表情变体；8 个变体必须有不同的视觉锚点，避免同义换词。
- 模板行使用 `编号. 具体表情变体；眉：……；眼：……` 格式；`具体表情变体` 必须与上方 8 个变体名一一对应。
- 模板字段以脸部锚点为主，可使用 `眉：`、`眼：`、`脸颊：`、`额头：`、`嘴：`、`牙齿：`、`舌头：`、`口边：`、`脸侧：` 等字段。
- 只写脸部卡通效果，不改身体、姿态、头部朝向、四肢动作、头发、衣服、饰品、背景或地面。
- 卡通效果只允许停留在脸部或脸侧极近区域；不得写成大面积污渍、真实伤口、血泊、衣物沾染、喷溅到画外或背景。
- 美式卡通感优先使用大眼、露齿、鼓嘴、吐舌、流口水、脸红、额头青筋、短烟线、旋涡眼、豆豆眼、下巴掉开、牙齿打颤等可一眼画出的视觉锚点。
- 单人模板的眼神优先绑定 `[目标物]` 或 `[证据物]`；多人模板可绑定 `[对方人物]`，也可同时回指 `[目标物]` 或 `[证据物]`。
- 1-2 为中等可见，3-4 为明显，5-6 为强烈，7-8 为夸张但仍合理。
- 每条至少有一处不对称；不要让 8 条只变成“眉眼嘴强度递增”的重复版本。

输出结构：

```md

适用场景：一句话说明何时使用。

具体表情变体：
1. 变体名
2. 变体名
3. 变体名
4. 变体名
5. 变体名
6. 变体名
7. 变体名
8. 变体名

**单人模板 1-8（看向证据物 / 目标物）**
1. 变体名；眉：……；眼：……；脸颊：……；嘴：……。

**多人模板 1-8（看向对方人物）**
1. 变体名；眉：……；眼：……；脸颊：……；嘴：……。
```

## 案例一：生气

### 生气

适用场景：被惹到、不服、争执、目标被破坏或对方行为太离谱。

具体表情变体：
1. 气到脸红
2. 咬牙爆筋
3. 气鼓鼓
4. 气到冒烟
5. 怒到露齿
6. 脸红憋气
7. 气到吐血短线
8. 笑不出来的怒脸

**单人模板 1-8（看向证据物 / 目标物）**
1. 气到脸红；眉：一侧眉头压低，另一侧眉尾挑起；眼：一侧眼瞪住[目标物]，另一侧眼收窄；脸颊：两侧脸颊明显涨红；嘴：嘴巴抿紧成下压斜线。
2. 咬牙爆筋；眉：双眉向眉心挤紧，一侧眉峰更高；眼：双眼怒视[证据物]；额头：额角鼓起一小段卡通青筋；牙齿：上下牙咬成锯齿线。
3. 气鼓鼓；眉：一侧眉尾下压，另一侧眉尾绷直；眼：一眼盯住[目标物]，另一眼半眯；脸颊：双颊鼓成气包；嘴：嘴巴闭紧成圆鼓形。
4. 气到冒烟；眉：双眉低压并形成尖锐折角；眼：双眼瞪向[证据物]，眼尾紧收；额头：额头上方冒两段短短卡通烟线；嘴：嘴角向下绷住。
5. 怒到露齿；眉：一侧眉心深深压下，另一侧眉峰上拱；眼：一侧眼锁住[目标物]，另一侧眼发硬；牙齿：一侧露出咬紧的牙齿；嘴：嘴角向一边拉紧。
6. 脸红憋气；眉：眉头向下挤住，眉尾明显发抖；眼：双眼憋红般瞪住[证据物]；脸颊：脸颊鼓起并泛红；嘴：嘴巴闭紧，像把话硬憋回去。
7. 气到吐血短线；眉：双眉大幅下压，一侧眉尾抽起；眼：双眼怒瞪[目标物]，眼神僵住；嘴：嘴巴猛地张开，吐出一小股卡通红色短线；口边：红色短线只停在嘴边，不扩散。
8. 笑不出来的怒脸；眉：一侧眉峰高高挑起，另一侧眉头压得很低；眼：一眼冷冷盯着[证据物]，另一眼半垂；嘴：嘴角想笑却僵住，牙齿露出一小段怒笑齿线。

**多人模板 1-8（看向对方人物）**
1. 气到脸红；眉：一侧眉头压低，另一侧眉尾挑起；眼：一侧眼瞪住[对方人物]，另一侧眼扫向[证据物]；脸颊：两侧脸颊明显涨红；嘴：嘴巴抿紧成下压斜线。
2. 咬牙爆筋；眉：双眉向眉心挤紧，一侧眉峰更高；眼：双眼怒视[对方人物]；额头：额角鼓起一小段卡通青筋；牙齿：上下牙咬成锯齿线。
3. 气鼓鼓；眉：一侧眉尾下压，另一侧眉尾绷直；眼：一眼盯住[对方人物]，另一眼扫回[目标物]；脸颊：双颊鼓成气包；嘴：嘴巴闭紧成圆鼓形。
4. 气到冒烟；眉：双眉低压并形成尖锐折角；眼：双眼瞪向[对方人物]，眼尾紧收；额头：额头上方冒两段短短卡通烟线；嘴：嘴角向下绷住。
5. 怒到露齿；眉：一侧眉心深深压下，另一侧眉峰上拱；眼：一侧眼锁住[对方人物]，另一侧眼发硬；牙齿：一侧露出咬紧的牙齿；嘴：嘴角向一边拉紧。
6. 脸红憋气；眉：眉头向下挤住，眉尾明显发抖；眼：双眼憋红般看向[对方人物]；脸颊：脸颊鼓起并泛红；嘴：嘴巴闭紧，像把反驳硬憋回去。
7. 气到吐血短线；眉：双眉大幅下压，一侧眉尾抽起；眼：双眼怒瞪[对方人物]，眼神僵住；嘴：嘴巴猛地张开，吐出一小股卡通红色短线；口边：红色短线只停在嘴边，不扩散。
8. 笑不出来的怒脸；眉：一侧眉峰高高挑起，另一侧眉头压得很低；眼：一眼冷冷看着[对方人物]，另一眼扫向[证据物]；嘴：嘴角想笑却僵住，牙齿露出一小段怒笑齿线。

## 案例二：喜欢

### 喜欢

适用场景：心动、宠爱、明显偏爱、被可见人物 / 动物 / 食物 / 礼物 / 目标物打动。

具体表情变体：
1. 爱心眼心动
2. 脸红偷看
3. 亮晶晶圆眼
4. 甜到眯眼
5. 捧场大笑脸
6. 舌尖小期待
7. 口水心动
8. 脸颊开花笑

**单人模板 1-8（看向证据物 / 目标物）**
1. 爱心眼心动；眉：双眉向外舒展开，一侧眉尾更高；眼：双眼变成清楚爱心形看向[目标物]；脸颊：脸颊泛起圆圆红晕；嘴：嘴巴开心张成小弧口。
2. 脸红偷看；眉：一侧眉尾轻轻挑起，另一侧眉尾压平；眼：一眼偷看[目标物]，另一眼害羞半垂；脸颊：靠近偷看方向的脸颊更红；嘴：嘴角藏着小小上扬。
3. 亮晶晶圆眼；眉：眉心完全放松，眉尾微微上抬；眼：双眼圆亮盯住[证据物]，瞳孔显得很大；嘴：嘴巴半开成惊喜小口。
4. 甜到眯眼；眉：双眉柔和放低，一侧眉尾轻弯；眼：双眼舒服眯起看向[目标物]；脸颊：脸颊鼓出甜甜弧度；嘴：嘴角软软向上弯。
5. 捧场大笑脸；眉：一侧眉峰高高抬起，另一侧眉峰舒展；眼：双眼弯成笑眼看向[证据物]；牙齿：露出整齐大白牙；嘴：嘴巴咧开成捧场大笑。
6. 舌尖小期待；眉：一侧眉尾期待抬起，另一侧眉尾放平；眼：一眼盯住[目标物]，另一眼眯起；舌头：舌尖轻轻露出；嘴：嘴巴张成小期待口。
7. 口水心动；眉：双眉明显向上拱起；眼：双眼黏住[目标物]，眼尾弯起；口边：嘴角挂一滴小口水；嘴：嘴巴半张，嘴角明显上扬。
8. 脸颊开花笑；眉：双眉高低分明地展开；眼：一眼看住[证据物]，另一眼开心弯起；脸颊：双颊红晕像圆形贴片；嘴：嘴巴张开成饱满笑弧。

**多人模板 1-8（看向对方人物）**
1. 爱心眼心动；眉：双眉向外舒展开，一侧眉尾更高；眼：双眼变成清楚爱心形看向[对方人物]；脸颊：脸颊泛起圆圆红晕；嘴：嘴巴开心张成小弧口。
2. 脸红偷看；眉：一侧眉尾轻轻挑起，另一侧眉尾压平；眼：一眼偷看[对方人物]，另一眼避开[证据物]；脸颊：靠近偷看方向的脸颊更红；嘴：嘴角藏着小小上扬。
3. 亮晶晶圆眼；眉：眉心完全放松，眉尾微微上抬；眼：双眼圆亮看向[对方人物]，瞳孔显得很大；嘴：嘴巴半开成惊喜小口。
4. 甜到眯眼；眉：双眉柔和放低，一侧眉尾轻弯；眼：双眼舒服眯起回应[对方人物]；脸颊：脸颊鼓出甜甜弧度；嘴：嘴角软软向上弯。
5. 捧场大笑脸；眉：一侧眉峰高高抬起，另一侧眉峰舒展；眼：双眼弯成笑眼看向[对方人物]；牙齿：露出整齐大白牙；嘴：嘴巴咧开成捧场大笑。
6. 舌尖小期待；眉：一侧眉尾期待抬起，另一侧眉尾放平；眼：一眼看住[对方人物]，另一眼扫向[目标物]；舌头：舌尖轻轻露出；嘴：嘴巴张成小期待口。
7. 口水心动；眉：双眉明显向上拱起；眼：双眼在[对方人物]和[目标物]之间发亮；口边：嘴角挂一滴小口水；嘴：嘴巴半张，嘴角明显上扬。
