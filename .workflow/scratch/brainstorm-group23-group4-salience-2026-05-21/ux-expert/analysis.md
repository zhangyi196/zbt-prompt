# UX Expert Analysis

## 结论

- 优先使用高轮廓变化、边界中断、结构不连续、表面完整性受损这类特征。
- 用户更容易先看到“缺口、裂痕、附着、断裂感”，而不是细纹、轻微偏移、弱装饰差。
- 差异应集中在单一局部，并与主体原有形态形成明确反差。

## 放宽方向

- 局部破坏性痕迹
- 局部附加 / 缺失
- 局部形态断开
- 局部遮挡残留

## 继续禁止

- 细小痕迹
- 低对比微变化
- 依赖颜色 / 材质才可见的变化
- 大面积纹理替换
- 整块区域改样式
- 影响大物体整体结构的改动

## 可落地规则语言

差异 MUST 第一眼可见，且仅限单个局部区域；优先制造轮廓或完整性突变，禁止扩散成大面积变化；差异面积应受控，但局部对比必须强，避免“看得见但找不到”。

## 防模式坍缩建议

不要写死少数范例，应改为“从局部破损、局部缺失、局部附着、局部断裂、局部异常连接等类别中任选一种”。强调“同类机制可多形态表达”，并要求每次只保留一个主差异信号。
