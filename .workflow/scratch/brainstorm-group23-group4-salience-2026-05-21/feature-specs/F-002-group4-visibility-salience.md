# F-002：组图 4 第一眼可见性增强

## 1. Requirements Summary

- `prompts/2.group-image/组图 4.md` MUST 继续保持 18 个差异点与九宫格 2×9 配额。
- 每个差异点 MUST 在“变化面积 ≥ 30% 或存在感变化”之外，再满足“第一眼可见”的更高准入标准。
- 差异 MUST 明显强于轻微变化，不得只靠细节纹理或弱变化成立。

## 2. Design Decisions

### 2.1 面积阈值不是终点

现有 30% 规则 SHOULD 保留，但只作为下限，不作为放行理由。若某候选点虽然面积达标，但仍需细看才能发现，则 MUST 淘汰。

### 2.2 优先强信号类型

组图 4 SHOULD 优先采用以下高可读变化机制：
- 明显增减
- 明显开合
- 明显缺失
- 明显反向 / 翻转
- 轮廓突变

图案内容变化仅在具备明显块状边界时才可使用。

### 2.3 图案变化的落点收紧

图案内容变化 SHOULD 只落在中小型独立承载物。地板、毛毯、整墙、整窗帘、大背景表面 MUST NOT 作为图案变化落点。

### 2.4 冷区与显著性兼容

优先填补冷区仍然成立，但不得为了冷区覆盖而放入弱差异。若某冷区只有轻微差异可做，MUST 改用更清晰的小实体变化或局部状态变化。

## 3. Interface Contract

建议后续编辑点：
- `总目标` 第 2 条显著性表述
- `空间分配规则` 中冷区补点策略
- `差异类型固定配额` 与 `类型定义` 中对图案变化的约束

## 4. Constraints & Risks

- 必须保持单一不同点原则。
- 必须保持大面积背景稳定。
- 风险：为追求显著性而过度牺牲卡通舒适感；应限制在“明确但不脏乱”的局部信号。

## 5. Acceptance Criteria

1. 每个差异点 MUST 无需放大即可识别。
2. 每个差异点 MUST 明显强于颜色微调、轻微缩放、微小偏移、弱纹理变化。
3. 图案变化 MUST 只落在中小型独立承载物，不得落在地板、毛毯等大块表面。
4. 若某点需要逐格比对才发现，则 MUST 判不通过。
5. 18 个差异点在增强显著性后，MUST 仍满足九宫格 2×9 与重叠回避。
