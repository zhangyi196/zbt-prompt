# Subject Matter Expert Analysis

## 差异类型分类框架

- **结构缺损型**：局部边界被切断、缺口、断裂，面积小但轮廓突变强。
- **表面破坏型**：局部出现撕裂、剥落、压塌、污染等破坏痕迹，需有明确形状边界。
- **功能失衡型**：局部位置、方向、连接关系异常，但主体身份不变。
- **状态反差型**：完整 / 受损、平整 / 扭曲、闭合 / 开裂等成对状态对比。

## 最需补强的显著性规则

### 组图 23 / List 2

差异 MUST 集中在主体中小区域，形成“单点高对比破坏”；边缘轮廓变化 SHOULD 先于纹理变化；禁止仅靠细线、轻微色差、零碎小痕迹成立。

### 组图 4

差异 MUST 以第一眼可见的形态突变为主，不可只满足幅度比例；应优先提升轮廓、开合、缺损、朝向异常等整体可读性。

## 必须避免的误伤方向

- 差异 MUST NOT 落在大面积背景或铺陈层。
- 差异 MUST NOT 扩散成大范围改面，导致主体被替换感。
- 规则 SHOULD NOT 给固定范式示例，应用“类别 + 强度约束”表达，避免模板化生成。
