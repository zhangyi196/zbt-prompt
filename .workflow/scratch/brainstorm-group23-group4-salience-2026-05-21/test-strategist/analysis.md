# Test Strategist Analysis

## 第一眼可见的可验证标准

- 差异点 MUST 在整张图首次扫视 1 秒内被定位，无需放大、回看或逐格比对。
- 差异点 MUST 具备清晰轮廓变化，优先形状、缺口、突起、方向、开合、增减。
- 中老年场景下，差异点 SHOULD 置于主体高关注区，且与周边元素边界明确。

## 如何区分明显 / 过大 / 过轻微

- 明显的小区域变化：MUST 为单点或单小块变化；面积受控，但轮廓改变量足够大。
- 面积太大：MUST NOT 改动主体大块结构、地板、毛毯、背景大片纹样。
- 过于轻微：MUST NOT 仅是细线、微痕、弱纹理、微角度、轻度位移或需细看才见的变化。

## 高价值验收项

### 组图 23
1. 每个差异项 MUST 服务“辅助差异一眼可见”，优先主体邻近的小部件形变。
2. 差异项 MUST 保持单点、小区域，不破坏主体完整识别。
3. 九宫格内差异分布 MUST 分散，紧邻位置 MUST NOT 同时出现互斥或近似差异。

### 组图 4
1. 每个差异项 MUST 明显强于“轻微变化”，不得仅靠细节纹理区分。
2. 差异类型 SHOULD 优先采用增减、开合、缺失、反向、轮廓突变。
3. 若某差异需逐格对照才发现，则该项 MUST 判不通过。
