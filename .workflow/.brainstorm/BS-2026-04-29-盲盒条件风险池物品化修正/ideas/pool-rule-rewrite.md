# Idea: 五层池“物品化”规则重写

## Problem

上一版 `conditional_items` 和 `blocked_or_risky` 的共同问题是：它们开始承载“关系、小附属件、痕迹、现象”，而不是具体物品。对游戏内容抽取来说，这会直接降低可用性，因为用户需要能放进画面、能圈选、能作为差异点的物品。

## Rule Set

### Universal Item Pool Rule

任何进入 `core_items`、`support_items`、`visible_small_items`、`conditional_items`、`blocked_or_risky` 的条目，都必须回答“这是一个具体物品吗？”和“它能被单独圈出来吗？”。

不通过的内容直接禁止进入五层池。

### conditional_items

定义：**条件启用的完整可见物品**，不是附属小标签。

进入条件：

- 具体物品或物品组。
- 中等以上可见体量。
- 可单独圈选。
- 只有在特定承载面、墙面、挂架、户外空间或装备关系存在时才合理。
- 不靠“贴在、夹在、套在、边缘、标签、卡片、扣件”等小附属关系成立。

### blocked_or_risky

定义：**具体但默认禁用的风险物品**。

进入条件：

- 必须是具体物品。
- 与场景相关，理论上常见。
- 因透明、反光、发光、动物本体、细绳/流苏、软布飘带、过度依赖光影或安全放置不稳定而禁用。

禁止进入：

- 折线、擦痕、阴影、高光、边线、气泡、沙粒、碎叶、微小污点。
- 任何“变化/痕迹/现象”而非物品的词。

## Suggested Data Split

短期保留字段名：

- `conditional_items`: 改内容，不改字段名。
- `blocked_or_risky`: 只保留具体风险物。

新增校验概念：

- `forbidden_patterns`: 非物品禁用词，用于测试和写库审查，不参与抽取。

## Example Replacement Direction

### 桌面+学习

conditional_items 可替换为：

- 桌面显示器支架
- 桌边文件挂篮
- 墙面课程白板
- 立式书包挂架
- 桌下移动书箱
- 书桌洞洞板
- 台面阅读灯罩盒
- 桌侧耳机挂架

blocked_or_risky 可保留为具体风险物：

- 透明亚克力尺
- 反光金属书签
- 发光台灯
- 细绳书签
- 流苏书签
- 玻璃墨水瓶
- 镜面笔筒
- 透明文件夹

移出五层池，进入 forbidden_patterns：

- 纸张边缘折线
- 微小擦痕
- 细小铅笔屑

### 海底+潜水

conditional_items 可替换为：

- 潜水装备挂架
- 水下标记浮标
- 海底采样箱
- 潜水绳收纳架
- 氧气罐固定架
- 水下工具背板
- 潜水相机支架
- 防水装备吊箱

blocked_or_risky 可保留为具体风险物：

- 鱼群本体
- 海龟本体
- 透明水母
- 发光珊瑚灯
- 反光贝壳盘
- 玻璃漂流瓶
- 细绳贝壳串
- 流苏海草挂饰

移出五层池，进入 forbidden_patterns：

- 细小气泡串
- 漂浮细海草丝
- 海底阴影斑
- 水面高光
- 微小沙粒

### 公园+野餐

conditional_items 可替换为：

- 折叠野餐椅
- 遮阳伞底座
- 野餐推车
- 户外置物架
- 野餐垫收纳筒
- 便携垃圾袋架
- 折叠餐桌
- 户外饮料桶

blocked_or_risky 可保留为具体风险物：

- 透明塑料杯
- 反光金属餐刀
- 细绳挂旗
- 流苏餐垫
- 玻璃饮料瓶
- 昆虫本体
- 远处飞鸟
- 镜面保温壶

移出五层池，进入 forbidden_patterns：

- 草地碎叶点
- 地面细小石子
- 风吹纸片边缘
- 野餐垫边线
- 微小污点
