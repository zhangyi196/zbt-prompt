# Refined Theme Names

## Recommended Format

Use natural scene names ending with `场景`, with an optional compact label for UI:

- Display name: `缝纫手作台场景`
- Short label: `手作台｜缝纫`
- Scene hint: `桌面、手作台、裁缝房、布料工作区`
