# Compact Space+Theme Names

## Rule

Use compact names:

`空间词+主题词`

The compact name is for display. The scene hint can live in documentation, hover text, or future UI help.

## Why This Fits

- Short enough for the tool UI.
- Still tells the user the spatial fit.
- Keeps theme flavor in the second half.
- Avoids vague style-only names like 复古、碎花、格纹.
