# Theme Rewrite Deep Dive

## Core Principle

盲盒主题应从“风格杂货词库”变成“明确场景词库”。主题味道来自真实场景组合，而不是大量 `复古/碎花/格纹/藤编/亚麻 + 盒/箱/篮/袋/挂/帘/串`。

## Theme Quality Criteria

一个合格主题需要满足：

1. 有明确场景：用户能想象它出现在桌面、柜架、地面、玄关、货架、户外等位置。
2. 有真实主物：至少 8-12 个一眼可识别、边界清楚的代表物。
3. 有配套小物：小物仍要可见，不以“迷你、标签、样片、签、扣、夹”凑数。
4. 有条件物：悬挂、灯、镜、透明、细线类单独低权重。
5. 有禁用尾词：每个主题列出容易批量滥用的尾词。
6. 有扩展空间：后续新增物品要符合主题，不靠同义容器扩容。

## Category Implication

即使后续暂时保留现有 `large / medium / small / hanging`，主题重写时也应按下面的真实含义填充：

- `large`: 主体锚点，不等于容器；可以是大件真实物或明确承载物。
- `medium`: 最稳定的清晰独立物，应成为主力类别。
- `small`: 小型但第一眼可见，不再大量使用迷你标签/样片/扣夹。
- `hanging`: 条件悬挂物，数量可少，不必硬凑 100。
