# Outdoor Category Expansion

## Problem

The reconstructed 16-category set was closer, but outdoor coverage was thin. It mainly covered:

- 庭院+园艺
- 门口+雨具
- 沙滩+度假
- 海底+潜水

It missed common target-image environments:

- park/grass
- campsite
- street/travel
- sports field

## Added Categories

| Category | Coverage | Example Object Types |
|---|---|---|
| 公园+野餐 | 公园草地、野餐垫、长椅、草坪 | 野餐篮、水壶、餐盒、毯子、风筝、球、纸袋 |
| 营地+露营 | 帐篷边、营地桌、户外装备区 | 手电筒、水壶、搪瓷杯、地图、背包、折叠椅、餐具 |
| 街道+出行 | 街边、车站、路口、出行随身物 | 背包、行李箱、帽子、票夹、水瓶、地图、购物袋 |
| 运动场+装备 | 操场、球场、健身区、户外运动角 | 球、水壶、毛巾、运动鞋、护腕、跳绳、奖牌 |

## Recommendation

Use 20 categories if the goal is broad scene coverage. Use 16 only if UI simplicity is more important than coverage.
