# Scene-Explicit Theme Names

## Naming Rule

Use names that directly tell the user where the blind-box theme fits:

`空间位置 + 主题用途 + 场景`

Examples:

- 手作台缝纫拼布场景
- 餐桌下午茶咖啡场景
- 厨房台面烘焙场景
- 玄关雨具出行场景

## Why This Is Better

- The theme itself tells the user when to use it.
- It keeps theme flavor without hiding the scene constraint.
- It discourages random object mixing across incompatible target images.
- It gives future item writers a clear spatial anchor.
