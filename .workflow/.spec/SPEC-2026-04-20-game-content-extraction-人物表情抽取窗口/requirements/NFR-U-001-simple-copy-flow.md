# NFR-U-001: 简洁复制流程

**Category**: Usability
**Priority**: Must

## Requirement

用户 SHOULD 能在 3 个主要动作内完成表情增强：粘贴输入、点击抽取、复制结果。

## Metric & Target

| Metric | Target | Measurement Method |
|--------|--------|--------------------|
| 主操作步骤 | <= 3 | 手工 UX 验证 |
| 错误提示语言 | 中文 | 错误路径检查 |
