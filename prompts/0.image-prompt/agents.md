# 图像提示词 Agent

本目录只负责绿圈洗图 AI 提示词链路；对应 skill 入口见 `../../skills/green-circle-photo-prompts/SKILL.md`。

## 入口

- `绿圈洗图 2 v1.2.1.1 3组.md`：文件夹路径驱动的洗图系统提示词，按参考图生成同名 `txt`。
- `绿圈洗图 2 v1.2.1.1 3组 版本管理.md`：记录该提示词版本变化；每次修改提示词后追加版本记录。
- `gpt洗图.md`：面向 gptimage2 的找不同游戏洗图提示词，将图片或英文插画提示词改写为固定模板下的中文主体描述。
- `gpt洗图 版本管理.md`：记录 `gpt洗图.md` 的版本变化；每次修改提示词后追加版本记录。

## 使用边界

- 本目录只维护绿圈洗图提示词，不负责主图、组图或桌面工具规则。
- 提示词更新后必须同步更新对应版本管理文档。
- 中文文件用 UTF-8；中文路径命令使用引号或 `-LiteralPath`。
