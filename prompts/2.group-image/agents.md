# 组图提示词 Agent

本目录只负责组图 AI 提示词链路；`组图 23 表情库.md` 继续放在仓库根目录，作为表情事实源供提示词和工具共用。

## 入口

- `组图 4.md`：18 个找不同差异点，九宫格每区 2 个；空间均匀分布为第 3 优先级，修改幅度 ≥30% 或存在感变化。30% 只是下限，第一眼可见才是最终准入条件；外部只输出 18 条具体修改内容。
- `组图 23.md`：Ref-A / Ref-B 到 Target 的差异迁移；List 1 使用内部剧情锚点表强化起因、关键道具、状态和结果的同链承载；List 1+2 九宫格覆盖率 ≥8/9，List 2 固定 18 条，并且只接收第一眼可见的小区域强信号；位置/修改禁颜色材质前缀，List 1 / List 2 禁止黑白相间条纹图案，List 2 禁止连续大表面承载颜色、材质、纹理、图案或结构破坏，使用内部锁定表逐对复扫相邻 / 同圈 / 同承载区冲突。
- `组图 23 表情前置.md`：人物表情类别前置规则；三组共享同一事实底座，只改变人物反馈角度；外部只输出 `负向`、`正向`、`正负向` 三个板块，摘要和自检只作内部检查。
- `../../组图 23 表情库.md`：人物表情模板库与单一事实源。
- `组图 4 版本管理.md`：记录 `组图 4.md` 的版本变化；每次修改提示词后追加版本记录。
- `组图 23 版本管理.md`：记录 `组图 23.md` 的版本变化；每次修改提示词后追加版本记录。

## 使用边界

- `组图 23 表情库.md` 不迁移、不拆分，默认保持根目录固定位置。
- 组图差异点仍遵守“一圈一物一变化”；不得依赖细线、小点、刻痕、碎屑、污渍、标签或价格签。
- 不改人物骨架、姿态、头身朝向、四肢动作、支撑关系和贴邻热区。
- 不做灯光、发光、明暗、阴影、高光、反光、滤光、投影或镜面成像变化；图案变化不落在地板、毛毯、整墙、整窗帘等大块表面。
- 禁止脑补不可见对象、画外空间、遮挡后区域或身份不稳定对象。
- 组图 23 List 1 + List 2 合并覆盖九宫格至少 8/9 区域（左上/中上/右上/左中/中心/右中/左下/中下/右下），每区域至少 1 点。
- 所有对象指代字段（位置/修改）禁止颜色词、材质词、纹理词作为对象前缀；VLM 颜色识别不稳定，带颜色描述易导致大面积错误差异点。
- 紧邻物体（间距小于一个物体宽度）不得同时设为差异点，审核阶段前置扫描紧邻对，输出前逐对复扫。
- 组图 4 差异点变化幅度不低于原物体 30% 或产生明显存在感变化（消失/新增/翻转/开合/图案替换），禁止仅颜色微调或轻微缩放。
