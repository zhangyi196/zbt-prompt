# 组图 23 表情库 v1.2.1

用途：为表情抽取 App 提供正向 + 负向共 50 个表情类别。每个表情含 8 个短模板变体，每个变体只写眉 / 眼 / 嘴，不改头部朝向、身体姿态、四肢动作、衣领、头发或耳饰。

使用规则：
- 正向 25 个、负向 25 个，保持 50% / 50%。
- 每次抽取时优先匹配剧情功能、人物关系和结局类型。
- 同一画面多人表情优先错开情绪极性和表情类别。
- 模板中的 `[剧情食物]`、`[剧情小物]`、`[证据物]`、`[目标物]`、`[对方人物]` 由后续流程按剧情替换成具体可见对象。
- `[对方人物]` 只用于眼部视线或眼神回指，不作为嘴部承载物。
- 每个表情的 1-4 条列为单人模板，眼部只看向或避开 `[证据物]` / `[目标物]`；5-8 条列为多人模板，眼部必须看向或避开 `[对方人物]`。
- 若剧情不支持嘴部承载物，删除占位物，改成闭紧、收住、细缝、扁张、卡住、发颤等面部状态。
