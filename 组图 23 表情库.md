# 组图 23 表情库 v1.3.0

用途：为表情抽取 App 提供正向 + 负向共 50 个表情类别。每个表情类别包含单人 8 条模板、多人 8 条模板；每条模板只写眉 / 眼 / 嘴，不改头部朝向、身体姿态、四肢动作、衣领、头发或耳饰。

使用规则：
- 正向 25 个、负向 25 个，保持 50% / 50%。
- 每次抽取时优先匹配剧情功能、人物关系和结局类型。
- 同一画面多人表情优先错开情绪极性和表情类别。
- 模板中的 `[剧情食物]`、`[剧情小物]`、`[证据物]`、`[目标物]`、`[对方人物]` 由后续流程按剧情替换成具体可见对象。
- `[对方人物]` 只用于眼部视线或眼神回指，不作为嘴部承载物。
- 每个表情的单人模板 1-8 只看向或避开 `[证据物]` / `[目标物]`；多人模板 1-8 必须看向、避开或回应 `[对方人物]`，也可同时回指 `[证据物]` / `[目标物]`。
- 模板强度从中等可见起步，1-2 为中等，3-4 为明显，5-6 为强烈，7-8 为夸张但合理；不保留轻微表情档。
- 若剧情不支持嘴部承载物，删除占位物，改成闭紧、收住、细缝、扁张、卡住、发颤等面部状态。
