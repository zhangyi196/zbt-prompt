# 组图 23 表情库 v1.0.1

用途：为表情抽取 App 提供正向 + 负向共 50 个表情类别。每个表情含 4 个短模板变体，每个变体只写眉 / 眼 / 嘴，不改头部朝向、身体姿态、四肢动作、衣领、头发或耳饰。

使用规则：
- 正向 25 个、负向 25 个，保持 50% / 50%。
- 每次抽取时优先匹配剧情功能、人物关系和结局类型。
- 同一画面多人表情优先错开情绪极性和表情类别。
- 模板中的 `[剧情食物]`、`[剧情小物]`、`[证据物]`、`[目标物]` 由后续流程按剧情替换成具体可见对象。
- 若剧情不支持嘴部承载物，删除占位物，改成闭紧、收住、细缝、扁张、卡住、发颤等面部状态。
