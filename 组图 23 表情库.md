# 组图 23 表情库 v2.1.0

用途：为表情抽取 App 提供正向 + 负向共 60 个表情类别。每个表情类别包含 8 个具体表情变体，并分别提供单人 8 条模板、多人 8 条模板；模板只写脸部表情字段，不改头部朝向、身体姿态、四肢动作、衣领、头发或耳饰。

使用规则：
- 正向 30 个、负向 30 个。
- 顶层类别保持简洁情绪词；具体表情变体写在每条模板开头，格式为 `编号. 具体表情变体；字段：……；字段：……`。
- 模板采用主视觉锚点优先写法：每条写 3-5 个最能体现该表情的脸部字段，不固定字段顺序，不强制写满眉 / 眼 / 脸颊 / 嘴 / 脸侧。
- 模板可使用 `眉：`、`眼：`、`脸颊：`、`额头：`、`嘴：`、`牙齿：`、`舌头：`、`口边：`、`脸侧：` 等脸部字段；每条至少包含 `眼：` 或 `嘴：` 之一，每个字段最多出现 1 次，不得写身体、姿态、头向、四肢、服装主体、发型或背景变化。
- 瞳孔内部的爱心、小星星、旋涡和小亮点属于眼部表情形状；不得扩散为脸外独立符号、背景装饰、真实发光、光晕或照明效果。
- 卡通红色短线、呕吐短线、烟线、黑线、汗滴、泪花等只能停留在嘴边、额头、脸侧或眼部极近区域，不得写成真实伤口、大面积污渍、喷溅到衣物 / 背景或画外。
- 允许使用脸部表情比喻，如小鸭嘴形瘪嘴、豆豆眼、星星瞳孔、爱心瞳孔、旋涡瞳孔、锯齿牙、下巴小幅掉开、花瓣红晕；不得写成真实变形、长出外来器官或动物部位。
- 模板中的 `[剧情食物]`、`[剧情小物]`、`[证据物]`、`[目标物]`、`[对方人物]` 由后续流程按剧情替换成具体可见对象。
- `[对方人物]` 只用于眼部视线或眼神回指，不作为嘴部承载物。
- 每个表情的单人模板 1-8 优先看向或避开 `[证据物]` / `[目标物]`；多人模板 1-8 必须看向、避开或回应 `[对方人物]`，也可同时回指 `[证据物]` / `[目标物]`。`犯困睡着` 允许闭眼或半闭眼，不强制眼神回指。
- 模板强度从中等可见起步，1-2 为中等，3-4 为明显，5-6 为强烈，7-8 为夸张但合理；不保留轻微表情档。
- 若剧情不支持嘴部承载物，删除占位物，改成闭紧、收住、细缝、扁张、卡住、发颤等面部状态。
