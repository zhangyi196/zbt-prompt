## 场景常识问题

1. 当前第四池把 `儿童房+玩具` 扩展成了儿童卧室生活用品库，安全条、门挡、洗衣篮、睡衣袋、拖鞋篮、雨伞筒等条目虽然真实存在，但不服务“玩具找不同”的主要场景。
2. `软木留言板`、`身高尺` 等固定墙面物不适合做可自由新增、删除、替换或移动的中型单物，且容易牵涉文字、刻度或墙面空间。
3. `温湿度计座`、`桌面风扇座`、`儿童水杯底座`、`地球仪底座` 等“底座 / 座”条目不是玩家一眼能当作独立玩具物品的视觉本体。
4. 车辆、轨道、积木、拼图、绘本、美术工具和娃娃餐厨玩具在多池反复出现，真实但密度过高，后续替换必须从不同视觉本体族补位，而不是继续换材质、主题或包装单位。
5. 未发现明确品牌 / IP 名称；`海盗帽`、`消防员头盔` 属于通用角色扮演道具，可保留少量，但不应作为核心池连续占位。
