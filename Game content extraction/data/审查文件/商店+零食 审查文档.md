## 总体结论

状态：`已复审并再次修改`

这类当前最主要的问题，不是零食不够多，而是很容易滑向两类伪多样性：

- `内容不同、包装壳不变`
  例如大量袋、盒、筒、罐只换内容物或口味。
- `前缀不同、服务壳不变`
  例如围绕杯、勺、纸巾、试吃物持续扩写出一串 `架 / 座 / 盘 / 垫 / 牌 / 板`。

本次复审确认，旧版本里这第二类问题已经重新出现在 `support_items` 和 `scene_expansion_items`：

- `support_items` 出现了 `纸杯分隔架 / 纸杯座 / 样品杯座 / 样品勺架 / 试吃签牌 / 防滑托垫 / 防漏杯垫 / 收银夹板` 一类弱本体条目。
- `scene_expansion_items` 虽然已经删掉一部分旧问题，但仍残留 `小票打印器 / 标签打印器` 这类不通过第四池边界检查的条目。
- `scene_expansion_items` 与其他池还出现了精确重叠：`牛皮纸袋`。

## 本次修改判断

本次修改不是继续在原模板里平移，而是按下面原则收紧：

- `support_items`
  保留真实的小辅件、取用件、封口件、试吃件；压掉“服务谁 + 架/座/垫/盘/板”的弱本体写法。
- `scene_expansion_items`
  继续收紧为前台自然中型物，优先保留顾客可见的结账物、饮品设备、提袋、冰桶、铁盒、少量自然装饰物。
- `跨池关系`
  第四池不得与前三池精确重叠；发现重复后必须替换，而不是解释其合理性。

## 本次主要替换点

- `support_items` 中，将明显的壳体化条目替换为更自然的小辅件，例如：
  `纸杯分隔架 -> 纸杯盖`
  `吸管展示夹 -> 纸吸管包`
  `纸杯座 -> 冷饮杯盖`
  `试吃签牌 -> 试吃牙签`
  `桌面餐巾架 -> 纸巾补充包`
  `样品杯座 -> 杯口封膜`
  `样品勺架 -> 木勺包`
  `防滑托垫 -> 硬币夹`
  `防漏杯垫 -> 纸杯套`
  `零食夹盘 -> 密封贴纸卷`
  `找零盘 -> 桌面记号笔`
  `收银夹板 -> 封箱胶带卷`

- `scene_expansion_items` 中，移除了不通过边界检查和跨池重叠的条目，例如：
  `小票打印器`
  `标签打印器`
  `牛皮纸袋`

- 同时把第四池从“打印器 / 清洁刷 / 套组 / 重复纸袋”继续拉回更自然的前台物，补入更稳定的对象族，例如：
  `点钞机 / 对讲机 / 果汁冷饮桶 / 咖啡保温桶 / 外带餐盒 / 帆布提袋 / 便携冷藏包 / 陶瓷花盆 / 鲜花花篮 / 留声机 / 点唱机`

## 本次压掉的根因

- `被服务对象不同，但稳定本体仍是架、座、垫、盘、板`
- `内容物不同，但稳定本体仍是袋、盒、筒、包`
- `第四池里混入过细服务物或与 support 重叠物`
- `为了过测试而留下的设备词，但放进当前场景后仍然不自然`

## 复审结果

- 四池均为 50 条，池内唯一。
- `scene_expansion_items` 与其他三池无精确重复。
- `scene_expansion_items` 已通过中型场景碎片边界自检。
- 第四池当前已不再包含 `小票打印器 / 标签打印器 / 牛皮纸袋` 这类本轮明确问题项。
- 本轮同时压降了 `support_items` 中成片出现的 `架 / 座 / 垫 / 牌 / 板` 弱本体条目。

## 保留风险

- `商店+零食` 天然仍会有较高的包装物密度，后续补写时仍需优先审查“内容换了、壳没变”的问题。
- 第四池当前仍保留一部分 `机 / 盒 / 袋 / 盘 / 筒` 尾词，这本身不构成问题；真正要防的是这些条目是否仍然在复写同一物体骨架。
- 如果后续继续补写，不应再回到 `展示架 / 价签牌 / 分隔架 / 样品座 / 防滑垫 / 记录板` 这一类说明成分包壳体的方向。
