# 桌面+学习 审查文档

## 1. 类别与范围

- 审查对象：`Game content extraction/data/blind_boxes.py` 中 `桌面+学习` 的四池内容。
- 本轮结论：当前版本比早期试点自然，但还不到可以直接收尾的状态；主要问题集中在第一池的书册横移、第二池的包装壳体扩写，以及第四池的包袋 / 挂图 / 教具板 / 摆件回流。
- 修改优先级：`scene_expansion_items` > `core_items` > `support_items` > `visible_small_items`。

## 2. 问题总览

- `core_items` 同时被 `书 / 本 / 册 / 词典 / 图鉴` 和 `板 / 盘 / 桶 / 盒 / 套` 两条模板线挤占，真正稳定的桌面主体物比例不够。
- `support_items` 里 `组 / 套 / 筒 / 盒 / 罐 / 匣 / 包` 过密，像在列包装壳和收纳壳，而不是在列辅助物本体。
- `visible_small_items` 格式基本合规，但纸类和书写耗材偏密，读感仍然偏“学习用品清单”。
- `scene_expansion_items` 是当前最大问题：包袋、盒壳、图板、挂图、学科教具和氛围摆件同时回流，已经偏离“前台自然单物”的第四池职责。

## 3. 按池问题与必换项

### core_items

- 书册横向平移过密，优先替换：`平装小说`、`图解百科书`、`自然图鉴`、`世界地图册`、`星座图册`、`双语词典`、`插图词典`、`方格练习簿`、`三孔活页本`、`练习题册`。
- 支撑壳和教具壳回流，优先替换：`风琴文件包`、`课程文件夹`、`几何绘图套`、`分子模型套`、`分数圆片托盘`、`彩色计数棒筒`、`计数方块桶`、`字母拼图板`、`世界拼图板`、`单词闪卡盒`、`活页纸箱`、`地图拼图盒`、`拼字方块桶`。
- 当前可作为锚点保留的方向：`厚页教材`、`精装参考书`、`桌面闹钟`、`科学计算器`、`图形计算器`、`平板阅读器`、`录音笔`、`桌面地球仪`、`等式天平`、`桌面画板`。

### support_items

- 包装壳和容器壳过密，优先替换：`马克笔筒`、`彩铅筒`、`蜡笔筒`、`直尺包`、`三角尺套`、`量角器套`、`圆规盒`、`橡皮擦罐`、`固体胶棒筒`、`长尾夹罐`、`回形针罐`、`活页环盒`、`便签抽取盒`、`笔芯筒`、`墨囊匣`、`文件护页包`、`活页分隔包`、`笔握套盒`。
- 问题根因不是这些物不能出现，而是这一池已经过度依赖“内容物 + 壳体”写法来凑多样性。
- 修改方向应回到真正被主体物使用的辅助工具本体，不再继续横向扩写各种包装单位。

### visible_small_items

- 整体格式合规，暂不建议整池推倒。
- 低优先级压缩纸面单位家族：`一包活页纸`、`一沓草稿纸`、`一包网格纸`、`一叠便签纸`、`一包彩色便条`、`一匣空白闪卡`、`一盒索引卡`、`一套薄练习本`、`一摞小笔记本`、`一卷包书纸`、`一卷护书膜`。
- 这些条目不是立刻必换项，但如果前两池不先压缩，本池会继续放大“纸本清单感”。

### scene_expansion_items

- 包壳和携带物回流，必须优先替换：`桌面纸巾盒`、`眼镜硬壳盒`、`护目镜盒`、`午餐保温袋`、`学生午餐盒`、`画具提包`、`帆布书包`、`水彩工具包`、`竖笛袋`、`字母印章盒`。
- 第二池工具回流，必须优先替换：`手摇削笔器`、`放大阅读尺`、`水彩调色盘`、`绘画水盆`、`色轮转盘`、`木质笔筒`。
- 教具图板和挂图横向平移，必须优先替换：`彩珠计数架`、`元素周期图`、`地球剖面图`、`人体骨骼模型`、`世界地图挂图`、`月相挂图`、`天文星图盘`、`地质分层图`。
- 氛围摆件和泛装饰偏多，建议压缩：`复古摆钟`、`打字机`、`音乐盒`、`万花筒`、`陶瓷小花盆`、`苔藓玻璃盆`。
- 当前较接近第四池方向的保留项：`折叠阅读架`、`桌面书立`、`斜面写字板`、`翻页台历`、`桌面月历`、`番茄计时器`、`沙漏计时器`、`指针计时钟`、`机械节拍器`、`点读机`、`复读机`、`小型显微镜`。

## 4. 多样性与模板密度

- 第一池最拥挤的两条模板线是 `书 / 本 / 册 / 词典 / 图鉴` 和 `板 / 盘 / 桶 / 盒 / 套`。
- 第二池最拥挤的尾词是 `组 / 套 / 筒 / 盒 / 罐 / 匣 / 包`，说明多样性过多依赖包装壳。
- 第三池格式没错，但 `纸 / 本 / 卡 / 便条` 家族已经成串，后续只宜小修，不宜继续加码。
- 第四池当前最危险的三条串联家族是 `包 / 袋 / 盒`、`图 / 挂图 / 盘`、`摆件 / 花盆 / 复古陈设`。

## 5. 第四池专项

- 第四池当前的根因不是单个坏词，而是写法又回到了“补一些像学习氛围的东西”。
- 结果是四类回流同时出现：收纳壳体、可携带包袋、学科图板、情绪摆件。
- 本类第四池应优先补“桌面前台一眼能认出的中型学习物、桌面辅助设备、阅读或计时相关稳定物”，而不是继续补包、盒、挂图、说明型模型板和泛装饰。
- 修改时建议先整段清掉高风险家族，再回填，不建议在现有坏家族上做微调。

## 6. 修改方向

- 第一池：保留少量高频书本锚点，把多余书册、纸本和教具壳让位给更像桌面主体物的具体对象。
- 第二池：把“内容物 + 壳体”写法改成真正的辅助工具本体，减少包装容器词。
- 第三池：维持 `一盒 / 一包 / 一套 / 一卷 / 一叠` 的可见单位格式，但拉开纸类、笔类、夹具类、标记类、绘画类的比例。
- 第四池：按“前台自然单物 > 中型边界 > 场景稳定性”回填，禁用 `包 / 袋 / 盒 / 图 / 挂图 / 说明型模型板 / 泛装饰摆件` 作为主要补位手段。
- 收尾时额外复查同后缀家族，不要只看唯一性和数量。

## 7. 本次修改记录

- 已基于上一轮结果完成第二次重做，修改文件：`Game content extraction/data/blind_boxes.py`。
- 本轮不再沿用“为了过规则再补一批 `xx板 / xx架 / xx器`”的写法，而是回到“真实桌面前台物优先”。
- `core_items`：
  继续清掉上一轮留下的弱本体，如 `文件立架`、`课程资料架`、`词卡架`、`学习平板支架` 等，改为更像真实桌面主体物或设备的 `台式显示器`、`电子记事本`、`便携显示器`、`桌面碎纸机`、`标签打印机`。
- `scene_expansion_items`：
  整段撤掉上一轮那批规则友好但读感生硬的弱模板条目，如 `课程记录板`、`标签排版板`、`试卷分区架`、`文件分区架`、`单词整理架`、`课程整理架`、`台式答题器`、`学习打卡器`、`口算训练器`、`阅读提示器` 等。
  改为以真实前台中型物为主的组合：少量真实 `架` 类，如 `折叠阅读架`、`写生画架`、`迷你画架`；台历与计时钟类，如 `翻页台历`、`番茄计时器`、`世界时区钟`；打印扫描与投影类，如 `便携投影机`、`桌面打印机`、`文件扫描器`；音频设备类，如 `桌面收音机`、`磁带录音机`、`黑胶唱机`；以及少量自然书册与桌面装饰物，如 `照片相册`、`艺术画册`、`皮面手账本`、`陶瓷小花盆`、`苔藓玻璃盆`。
- 本轮核心目标不是继续“换壳过审”，而是把第四池从抽象功能词重新拉回能被普通人一眼认出的桌面场景物。

## 8. 修改后复审

- 已完成本类第二轮结构复查。
- `桌面+学习` 四池当前均为 50 条，且池内唯一。
- `scene_expansion_items` 与其他三池无机械精确重复。
- 按本类专项口径复查：
  `scene_expansion_items` 50 条全部通过中型边界检查；
  本类四池均未命中 blocked patterns；
  第四池与其他三池无跨池机械重复。
- 人审结论：
  相比上一轮，这次已经明显去掉了 `课程 / 试卷 / 学习 + 板 / 架 / 器` 的假修复链条；
  当前第四池虽然仍有设备类密度，但整体已经更像真实桌面前台，而不是临时拼出来的规则产物；
  当前版本可以作为 `桌面+学习` 的新基线继续使用。

## 9. 剩余风险与暂留理由

- 本轮已经解决的根因是“把旧坏词换成另一批弱模板词”；这一点比上一轮有实质改善。
- 仍保留两类中低风险：
  `core_items` 里的书册和学习设备仍偏多，但已经比早期更分层；
  `scene_expansion_items` 里的 `机 / 钟 / 册 / 本` 家族仍有一定密度，只是已经明显好于 `板 / 架 / 器` 的抽象职能串联。
- 这些残余风险本轮暂留的原因是：
  先优先修正读感最差的弱本体链条，避免为了继续压尾词而再次引入新的伪自然词；
  `桌面+学习` 本身允许一定比例的设备、计时和书册物，完全压平反而容易失真。
- 如果下一轮继续细修，本类建议继续轻压第四池的 `机 / 钟 / 册 / 本` 密度，但不要回退到 `课程 / 试卷 / 学习 + 板 / 架 / 器` 这类说明词 + 载体词方案。
