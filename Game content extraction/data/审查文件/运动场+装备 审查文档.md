# 运动场+装备 审查文档

## 1. 类别与范围

- 审查对象：`Game content extraction/data/blind_boxes.py` 中 `运动场+装备` 的四池内容。
- 本轮结论：当前版本的第一池仍能读出“运动场装备区”的主体场景，但已经混入较多计分、补给和裁判服务物；第二池被 `包 / 盒 / 篮 / 盘 / 架 / 带 / 卷` 的补充壳和护理壳拉散；第四池明显没有通过，已经被写成场边收纳、支架、补给和裁判服务系统的混合池。
- 修改优先级：`scene_expansion_items` > `support_items` > `core_items` > `visible_small_items`。

## 2. 问题总览

- `core_items` 的球类、球拍、训练器材、护具和力量器材主场景还在，但“场边服务区”和“补给计分区”的读感已经开始抬头。
- `support_items` 的主要问题不是与运动场无关，而是把真实辅助工具、护理件、收纳壳、补给壳和记录物混在了一池里。
- `visible_small_items` 格式合规，但能量补给、冰敷护理、裁判小件和一次性消耗品成串后，已经很像“赛事补给包”。
- `scene_expansion_items` 是当前最大问题：架、箱、筐、桶、托盘、支架、挂架和夹板集中回流，明显偏离“前台自然中型物”的第四池职责。

## 3. 按池问题与必换项

### core_items

- 本池最大问题不是整体失场，而是 `板 / 盘 / 包 / 牌 / 饮` 这类场边服务物已经开始挤占主体位。
- 低优先级平衡项：`计分翻牌板`、`补给托盘`、`急救包`、`计时秒表`、`瓶装电解质饮`、`边线旗`、`换人牌`。
- 这些条目不一定要立刻整片清洗，但它们说明本池已经开始用“裁判物、补给物和场边服务物”维持密度；后续若要腾位，可优先从这组压缩。
- 当前可作为锚点保留的方向：`篮球`、`足球`、`排球`、`橄榄球`、`手球`、`发球机`、`起跑器`、`网球拍`、`羽毛球拍`、`乒乓球拍`、`匹克球拍`、`棒球棒`、`曲棍球杆`、`高尔夫球杆`、`棒球手套`、`守门员手套`、`拳击手套`、`护胫板`、`训练标志锥`、`敏捷梯`、`跳绳`、`瑜伽垫`、`训练垫`、`泡沫轴`、`球篮`、`球筐`、`哑铃`、`壶铃`、`负重沙袋`、`重量背心`、`飞盘`、`运动背包`、`装备球袋`、`球鞋`、`运动水壶`、`护膝`、`护肘`、`护腕`、`护踝`、`裁判旗`、`运动毛巾`、`训练标志盘`、`训练豆袋`。

### support_items

- 本池被护理壳、收纳壳和记录壳拉得过散，优先替换：`球网收纳包`、`记分夹板`、`战术白板`、`标志盘提篮`、`标志锥提篮`、`护齿盒`、`护具收纳包`、`毛巾夹包`、`补给冰袋`、`食品保冷包`、`运动补给罐`、`电解质粉罐`、`擦汗毛巾卷`、`裁判旗`、`手持记分器`、`口哨盒`、`训练记录本`、`便携垃圾桶`、`衣物篮`、`鞋带盒`、`鞋垫盒`、`急救药盒`、`折叠坐垫`、`便携风扇`。
- 本池里也存在“同对象不同补位 + 相同壳体”的伪多样性，优先复查：`冷敷冰袋`、`冰敷绑带`、`运动胶带卷`、`肌贴卷`、`弹性绷带卷`、`手腕护带`、`膝部护带`、`脚踝护带`。
- 问题根因不是这些词完全不能出现，而是这一池已经同时承担了“真实辅助工具池”“护理补给池”和“裁判记录收纳池”三种职责。
- 当前更稳的辅助物方向仍应放在真正参与充气、测压、训练摆放、拉伸、防护、清洁和裁判动作的工具本体，如 `手动球泵`、`球压计`、`训练秒表`、`战术记号笔`、`折叠跨栏`、`训练圆环`、`训练绳梯`、`拉伸绳`、`握力器`、`水壶提带`、`水壶杯套`、`清洁喷瓶`、`球鞋刷`、`水洗海绵`。

### visible_small_items

- 整体格式合规，暂不建议整池推倒。
- 低优先级压缩项：`一盒创可贴`、`一包冷敷贴`、`一袋冰袋`、`一盒护齿套`、`一套护腕`、`一套护踝`、`一包吸汗头带`、`一包吸汗发带`、`一盒能量棒`、`一排能量饮料`、`一包电解质粉`、`一筒电解质片`、`一盒葡萄糖块`、`一包盐味脆饼`、`一袋混合坚果`、`一盒果泥包`、`一包纸杯`、`一包纸吸管`、`一包木质小叉`、`一叠餐巾纸`、`一卷垃圾袋`、`一盒保冷冰袋`、`一盒防晒霜小管`、`一排护手霜小管`、`一包酒精棉片`、`一套口哨`、`一套秒表`、`一盒记分笔`、`一包擦汗毛巾`、`一盒彩色臂带`、`一盒湿巾`。
- 这些条目本身未必违规，但它们和前两池、第四池的补给、护理和裁判壳叠加后，会继续放大“运动场边补给区”的读感。

### scene_expansion_items

- 架、座、支架和挂架家族回流，必须优先替换：`球衣挂架`、`毛巾挂架`、`号码牌架`、`计时牌架`、`角旗座`、`伞座`、`记分牌架`、`运动饮水架`、`毛巾收纳架`、`场边水杯架`、`计分板支架`。
- 箱、筐、篮、桶、托盘和车体家族过密，必须优先替换：`轮式器材车`、`器材周转箱`、`训练标志桶`、`球员补给箱`、`冰袋收纳箱`、`补水筐`、`补给保温箱`、`场边垃圾桶`、`换鞋收纳筐`、`能量饮料箱`、`场边储物筒`、`器材托盘`、`便携水桶`、`球员饮水桶`、`场边收纳篮`、`便携收纳车`、`运动整理箱`。
- 夹板、夹件和标识服务物回流，必须优先替换：`战术夹板`、`战术磁贴盘`、`比赛指示牌`、`裁判记号夹`、`队牌夹`、`裁判卡夹`、`场地标识牌`、`裁判夹板`、`场边标志盘座`。
- 场边服务器具和近义重复项混入，必须整体压缩：`便携长椅`、`折叠换鞋凳`、`场边折叠椅`、`场边隔离栏`、`场边遮阳伞`、`场边风扇`、`场边喷雾壶`、`毛巾卷筒`、`替补席靠垫`、`替补席坐垫`、`运动护具袋`、`证件挂袋`、`裁判口哨绳`。
- 当前较接近第四池方向、可作为保留候选的条目相对有限：`便携长椅`、`折叠换鞋凳`、`场边折叠椅`、`场边隔离栏`、`场边遮阳伞`。

## 4. 多样性与模板密度

- 第一池当前最明显的压力不是“装备主体物太多”，而是 `板 / 盘 / 包 / 牌 / 饮` 与补给、计分、裁判家族一起抬高了场景杂音。
- 第二池最拥挤的尾词是 `包 / 盒 / 篮 / 盘 / 架 / 带 / 卷`，主要问题是“不同被服务对象 + 相同护理壳/收纳壳”的伪多样性。
- 第三池格式没错，但 `护理耗材 / 能量补给 / 裁判小件 / 一次性消耗品` 已经成串，后续只宜轻压，不宜继续扩写。
- 第四池当前最危险的四条串联家族是 `架 / 座 / 支架 / 挂架`、`箱 / 筐 / 桶 / 托盘 / 收纳篮`、`夹板 / 指示牌 / 记号夹 / 卡夹`、`风扇 / 喷雾壶 / 靠垫 / 护具袋` 这几类场边服务壳体链。

## 5. 第四池专项

- 第四池当前的根因不是个别坏词，而是整池写法站在“场边收纳、补给和裁判服务系统”的视角，而不是“运动场前台场景补充物”的视角。
- 结果是四类问题同时叠加：支架挂架成串、箱筐桶盘成串、夹板和标识服务物成串、风扇喷雾和靠垫等场边服务器具混入。
- 本类第四池应优先补“普通玩家一眼就能认出的场边休息区、换鞋区、等待区或比赛边线附近”的自然中型单物，而不是继续补支架、收纳箱、补给桶和裁判服务壳。
- 这一池不适合沿现有词表微调；建议修改阶段按高风险家族整段清理，再回填更自然的前台物。

## 6. 修改方向

- 第一池：保留球类、球拍、训练器材、护具、力量器材和典型比赛装备这些稳定主体，必要时压缩计分、补给和裁判服务物的横向平移。
- 第二池：从“护理补给壳与裁判记录壳混池”拉回“真正辅助工具池”，优先保留充气、测压、训练摆放、拉伸、防护和简单清洁动作里会直接上手的物件本体。
- 第三池：继续保持 `一盒 / 一包 / 一袋 / 一排 / 一套 / 一叠 / 一卷 / 一筒` 的可见单位格式，但避免护理耗材、能量补给、裁判小件和一次性消耗品继续膨胀。
- 第四池：按“运动场前台自然单物 > 场景一眼可读 > 中型边界”重建，禁用支架、收纳箱、补给桶、夹板和场边服务器具作为主要补位手段。
- 收尾时额外检查同后缀家族，尤其是 `架 / 座 / 箱 / 桶 / 盘 / 篮 / 包 / 板` 是否还成片出现。

## 7. 本次修改记录

- 本轮实际修改了 `blind_boxes.py` 中 `运动场+装备` 的 `support_items` 和 `scene_expansion_items`，并顺手修正了第一池中命中禁词的 `边线旗`，改为 `线审旗`。
- 第二池的修改方向是把“护理补给壳、收纳壳和裁判记录壳混池”拉回“真实充气、测压、训练摆放、拉伸、防护和简单清洁工具池”。
- 具体清掉了 `球网收纳包 / 记分夹板 / 战术白板 / 标志盘提篮 / 标志锥提篮 / 护齿盒 / 护具收纳包 / 毛巾夹包 / 补给冰袋 / 食品保冷包 / 运动补给罐 / 电解质粉罐 / 擦汗毛巾卷 / 手持记分器 / 口哨盒 / 训练记录本 / 便携垃圾桶 / 衣物篮 / 鞋带盒 / 鞋垫盒 / 急救药盒 / 折叠坐垫 / 便携风扇` 这类弱本体壳层。

- 第四池本轮按高风险家族整段清理，移除了 `支架 / 挂架 / 收纳箱 / 水桶 / 托盘 / 夹板 / 指示牌 / 风扇 / 喷雾壶 / 靠垫 / 护具袋` 这类场边收纳、补给和裁判服务系统写法。
- 第四池回填改成“时间物 + 小型环境设备 + 场边地面物 + 器材停靠基座 + 少量赛事画册图册”结构，尽量让它更像运动场前台自然可见的中型场景补充物，而不是场边后勤区。
- 本轮又继续压缩了 `地毯卷 / 毯面 / 席面 / 花盆 / 图册 / 画册` 这几组高频尾词，把它们都收敛到“每种最多两个”，并用 `球门配重座 / 训练标杆座 / 跨栏摆放座 / 看台围栏座 / 跑道围栏座 / 备用球篮 / 训练球篮 / 球网配重座 / 热身分区 / 鞋底吸水擦` 这类更贴近运动场等待区、换鞋区和边线附近的中型物回填。

## 8. 修改后复审

- 当前四池数量均为 50，且池内唯一。
- 第四池与其余三池无重叠。
- 四池本轮本地复查未命中禁词，且第四池全部通过 `_is_medium_scene_fragment_item` 的中型场景边界校验。
- 第四池中的 `地毯卷 / 毯面 / 席面 / 花盆 / 图册 / 画册` 当前实值分别为 `2 / 2 / 2 / 2 / 2 / 2`，已经不再出现同后缀连续堆叠压读感的问题。
- 这说明本轮已经先把“结构性错误”修掉了，尤其是第四池不再被 `架 / 座 / 箱 / 桶 / 盘 / 篮 / 包 / 板` 或 `地毯卷 / 毯面 / 席面 / 花盆 / 图册 / 画册` 这类串联家族主导。
- 从读感上看，第二池已经明显回到运动辅助工具池；第四池也从“收纳、支架、补给与裁判服务系统”转回“运动场前台场景补充物”。
- 本节结论仅针对 `运动场+装备` 的定点复查，不代表全仓所有类别已全量通过。

## 9. 剩余风险与暂留理由

- 第一池里仍保留了少量补给物、裁判物和场边服务物，例如 `计分翻牌板`、`补给托盘`、`急救包`、`计时秒表`、`瓶装电解质饮`、`换人牌`；这轮暂不动，是为了先稳住主体池，不把问题从第四池转移到第一池。
- 第三池仍保留了较多护理耗材、能量补给、裁判小件和一次性消耗品，例如 `一盒创可贴`、`一包冷敷贴`、`一袋冰袋`、`一盒能量棒`、`一排能量饮料`、`一包电解质粉`、`一筒电解质片`、`一套口哨`、`一套秒表`、`一盒记分笔`；它们当前格式合规，但后续如果继续做第三池精修，可以再轻压一次。
- 第四池虽然已经脱离“场边后勤壳链”，也压住了 `地毯卷 / 毯面 / 席面 / 花盆 / 图册 / 画册` 的成串问题，但现在仍有 `钟 / 历 / 机 / 盆 / 座 / 篮 / 图 / 册` 这些自然家族；后续若继续微调，重点应放在压模板密度，而不是重新引入 `架 / 座 / 箱 / 桶 / 盘 / 篮 / 包 / 板`。
- 当前版本已经可以作为后续同类修改的合格基线，下一步若继续优化，建议优先做“第一池补给裁判物轻压”和“第三池护理补给单位轻压”，而不是再大改第二池和第四池。
