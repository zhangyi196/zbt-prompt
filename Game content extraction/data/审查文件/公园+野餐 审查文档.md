# 公园+野餐 审查文档

## 1. 类别与范围

- 审查对象：`Game content extraction/data/blind_boxes.py` 中 `公园+野餐` 的四池内容。
- 本轮结论：当前版本的第一池仍能读出“草地野餐区”的主体场景，但已经混入较多餐盒、拼盘、图册、棋盒和留言册；第二池被 `包 / 杯 / 盘 / 盒 / 罐 / 卷 / 套` 的补充壳和分装壳拉散；第四池明显没有通过，已经被写成阅读写生区、记录展示区和分装提篮区的混合池。
- 修改优先级：`scene_expansion_items` > `support_items` > `core_items` > `visible_small_items`。

## 2. 问题总览

- `core_items` 的野餐垫、保冷包、篮子、水壶、草地游戏和休闲物主场景还在，但“餐食摆盘区”和“阅读娱乐区”的读感已经开始抬头。
- `support_items` 的主要问题不是与野餐无关，而是把真实辅助件、餐食分装壳、杯盘壳、收纳壳和消耗品混在了一池里。
- `visible_small_items` 格式合规，但零食、小食、饮料小瓶和一次性餐具成串后，已经很像“野餐补给包”。
- `scene_expansion_items` 是当前最大问题：本、册、板、盒、盘、篮、提篮、画架和颜料盘集中回流，明显偏离“前台自然中型物”的第四池职责。

## 3. 按池问题与必换项

### core_items

- 本池最大问题不是整体失场，而是 `篮 / 盒 / 盘 / 壶 / 册` 壳层和阅读娱乐配件已经开始挤占主体位。
- 低优先级平衡项：`双层便当盒`、`分格午餐盒`、`三明治拼盘`、`水果拼盘`、`奶酪拼盘`、`面包篮`、`沙拉碗`、`蔬菜蘸酱盘`、`曲奇铁盒`、`饼干罐`、`果汁玻璃瓶`、`饮料分装桶`、`草地跳棋盒`、`野餐留言册`、`扑克牌盒`、`小号冰桶`、`户外棋盘盒`、`自然图鉴`、`折叠画板`。
- 这些条目不一定要立刻整片清洗，但它们说明本池已经开始用“摆盘壳、分装壳和休闲读物”维持密度；后续若要腾位，可优先从这组压缩。
- 当前可作为锚点保留的方向：`格纹野餐垫`、`防潮野餐毯`、`折叠坐垫`、`充气靠枕`、`藤编野餐篮`、`帆布保冷包`、`硬壳冷藏箱`、`帆布手提袋`、`折叠扇`、`柠檬水壶`、`冷泡茶壶`、`热饮保温壶`、`保温水瓶`、`搪瓷马克杯`、`法棍面包`、`贝果面包`、`纸杯蛋糕`、`迷你三明治`、`手持蒲扇`、`野花花束`、`宽檐遮阳帽`、`便携音箱`、`草地飞碟`、`软式草地球`、`羽毛球拍`、`泡泡机`、`风筝线盘`、`拍立得相机`、`双筒望远镜`、`折叠阳伞`。

### support_items

- 本池被餐食分装壳、杯盘壳和消耗品壳拉得过散，优先替换：`野餐餐具包`、`木质餐盘`、`折叠餐杯`、`吸管杯`、`保温杯套`、`食物罩`、`保鲜盖`、`餐盘提架`、`调味分装盘`、`水果叉套`、`调味瓶组`、`胡椒研磨罐`、`盐粒罐`、`餐巾卷`、`湿巾包`、`纸巾包`、`擦手毛巾卷`、`便携洗手液`、`手部消毒凝胶`、`小号急救包`、`野餐垃圾袋`、`保鲜膜卷`、`冷藏箱隔板`、`野餐篮内衬`、`保冷包肩带`、`扑克牌套`、`飞盘收纳套`、`羽毛球筒`、`风筝提线盘`、`坐垫束带`、`野花扎带`、`纸杯提架`。
- 本池里也存在“同对象不同补位 + 相同壳体”的伪多样性，优先复查：`软木杯垫`、`饮料龙头`、`保冷冰袋`、`食品密封夹`、`折叠垃圾桶`。
- 问题根因不是这些词完全不能出现，而是这一池已经同时承担了“真实辅助工具池”“餐食补给分装池”和“娱乐收纳配件池”三种职责。
- 当前更稳的辅助物方向仍应放在真正参与开瓶、分食、切取、夹取、压角、保冷和清洁动作的工具本体，如 `小号冰夹`、`开瓶器`、`切片砧板`、`面包夹`、`沙拉夹`、`桌布压石`、`水果刀`、`蜂蜜勺`、`防晒霜管`、`驱蚊喷雾瓶`、`泡泡液瓶`、`遮阳帽夹`、`野餐围裙`。

### visible_small_items

- 整体格式合规，暂不建议整池推倒。
- 低优先级压缩项：`一盒迷你三明治`、`一盒奶酪块`、`一盒水果块`、`一盘草莓`、`一盘葡萄`、`一碗蓝莓`、`一盘小番茄`、`一盘蔬菜条`、`一盘曲奇饼干`、`一盘椒盐脆饼`、`一碗爆米花`、`一碗坚果`、`一碗橄榄`、`一碗水果沙拉`、`一碗意面沙拉`、`一盒纸杯蛋糕`、`一袋薯片`、`一袋爆米花`、`一袋面包卷`、`一袋谷物棒`、`一包果干`、`一包饼干条`、`一排迷你水瓶`、`一排果汁小瓶`、`一排苏打罐`、`一排酸奶杯`、`一排果酱小罐`、`一排蜂蜜小瓶`、`一排调味小罐`。
- 这些条目本身未必违规，但它们和前两池、第四池的餐食分装壳叠加后，会继续放大“野餐零食与补给区”的读感。

### scene_expansion_items

- 图册、本册、记录板和拼贴板家族回流，必须优先替换：`儿童绘本`、`户外故事本`、`观鸟图册`、`花草拼贴板`、`速写本`、`路线记录板`、`植物图鉴`、`昆虫图鉴`、`折页地图册`。
- 计分器、录音笔、显微镜和观察器具回流，必须优先替换：`野餐计分器`、`便携显微镜`、`儿童显微镜`、`便携录音笔`、`电子记分器`。
- 盘、篮、托盘、提篮和盒体过密，必须优先替换：`玻璃冰桶`、`木质托盘`、`草编提篮`、`水果浅篮`、`饮料提篮`、`搪瓷餐盘`、`抽纸盒`、`随身湿巾盒`、`彩色粉笔盒`、`点心铁盒`。
- 写生和涂鸦区器具过密，必须优先替换：`水彩笔筒`、`涂鸦蜡笔筒`、`写生画架`、`水彩颜料盘`、`观景画板`。
- 后场服务物与功能器具混入，必须整体压缩：`户外驱蚊器`、`手持喷雾器`、`喷雾降温器`、`急救药包`、`擦手布卷`、`野餐毛巾卷`、`户外餐垫`、`花纹靠垫`、`野餐坐垫`、`风筝卷轴`。
- 当前较接近第四池方向、可作为保留候选的条目相对有限：`折叠棋盘`、`拼字游戏板`、`拼图板`、`飞盘游戏盘`、`豆袋投掷板`、`磁性飞镖盘`、`即时成像相机`、`便携收音机`、`手摇泡泡机`、`野花花篮`。

## 4. 多样性与模板密度

- 第一池当前最明显的压力不是“野餐物太多”，而是 `篮 / 盒 / 盘 / 壶 / 册` 与餐食摆盘家族、娱乐读物家族一起抬高了场景杂音。
- 第二池最拥挤的尾词是 `包 / 杯 / 盘 / 盒 / 罐 / 卷 / 套`，主要问题是“不同被服务对象 + 相同分装壳/收纳壳”的伪多样性。
- 第三池格式没错，但 `小食盒 / 小瓶 / 小罐 / 一次性餐具 / 零食小包` 已经成串，后续只宜轻压，不宜继续扩写。
- 第四池当前最危险的四条串联家族是 `本 / 册 / 图鉴 / 记录板`、`托盘 / 餐盘 / 浅篮 / 提篮 / 盒`、`笔筒 / 画架 / 颜料盘 / 画板`、`喷雾器 / 驱蚊器 / 急救包 / 毛巾卷` 这几类野餐服务壳体链。

## 5. 第四池专项

- 第四池当前的根因不是个别坏词，而是整池写法站在“草地阅读写生与野餐服务系统”的视角，而不是“公园前台场景补充物”的视角。
- 结果是四类问题同时叠加：阅读图册和记录本成串、提篮托盘和餐盘成串、写生涂鸦器具成串、喷雾器和急救包等服务物混入。
- 本类第四池应优先补“普通玩家一眼就能认出的草地野餐区、摆餐区、休闲互动区”里的自然中型单物，而不是继续补图鉴、记录板、画架、笔筒、托盘和点心盒。
- 这一池不适合沿现有词表微调；建议修改阶段按高风险家族整段清理，再回填更自然的前台物。

## 6. 修改方向

- 第一池：保留野餐垫、保冷包、野餐篮、水壶、草地游戏和休闲主体，必要时压缩拼盘、盒罐、图册和棋盒的横向平移。
- 第二池：从“餐食补给壳与娱乐收纳壳混池”拉回“真正辅助工具池”，优先保留开瓶、切取、夹取、压角、分食、保冷和防护动作里会直接上手的物件本体。
- 第三池：继续保持 `一盒 / 一盘 / 一碗 / 一袋 / 一包 / 一排 / 一筒 / 一套 / 一叠 / 一卷` 的可见单位格式，但避免零食、小瓶、一次性餐具和小罐继续膨胀。
- 第四池：按“公园前台自然单物 > 场景一眼可读 > 中型边界”重建，禁用图鉴、本册、记录板、托盘、提篮、画架、笔筒和服务器具作为主要补位手段。
- 收尾时额外检查同后缀家族，尤其是 `本 / 册 / 板 / 盒 / 篮 / 盘 / 筒 / 卷 / 包` 是否还成片出现。

- 第二池的修改方向是把“餐食分装壳、杯盘壳、消耗品壳和娱乐收纳壳混池”拉回“真实开瓶、切取、夹取、压角、保冷、清洁和草地休闲辅助工具池”。

## 8. 修改后复审

- 当前四池数量均为 50，且池内唯一。
- 第四池与其余三池无重叠。
- 第四池本轮本地复查未命中禁词，且全部通过 `_is_medium_scene_fragment_item` 的中型场景边界校验。
- 这说明本轮已经先把“结构性错误”修掉了，尤其是第四池不再被 `本 / 册 / 板 / 盒 / 篮 / 盘 / 筒 / 卷 / 包` 这类阅读写生和分装摆餐壳主导。
- 从读感上看，第二池已经明显回到野餐辅助工具池；第四池也从“阅读写生、分装摆餐和服务补给系统”转回“公园前台场景补充物”。
- 进一步按后缀统计复查后，第四池里的 `卷 / 席面 / 毯面 / 布面` 已分别压到 `2 / 2 / 2 / 1`，满足这一轮补充要求。
- 当前再次执行 `test_blind_box_content_model.py` 时，全量 14 项规则测试已通过；本类本轮微调没有引入新的共享规则回归。
- 本节结论仅针对 `公园+野餐` 的定点复查，不代表全仓所有类别已全量通过。

## 9. 剩余风险与暂留理由

- 第一池里仍保留了少量摆盘壳、分装壳和休闲读物，例如 `双层便当盒`、`分格午餐盒`、`三明治拼盘`、`水果拼盘`、`奶酪拼盘`、`面包篮`、`沙拉碗`、`蔬菜蘸酱盘`、`曲奇铁盒`、`饼干罐`、`饮料分装桶`、`草地跳棋盒`、`野餐留言册`、`扑克牌盒`、`户外棋盘盒`、`自然图鉴`、`折叠画板`；这轮暂不动，是为了先稳住主体池，不把问题从第四池转移到第一池。
- 第三池仍保留了较多小食、小瓶和一次性餐具单位，例如 `一盒迷你三明治`、`一盒奶酪块`、`一盒水果块`、`一盘草莓`、`一盘葡萄`、`一排迷你水瓶`、`一排果汁小瓶`、`一排苏打罐`、`一排酸奶杯`、`一排果酱小罐`、`一排蜂蜜小瓶`、`一排调味小罐`；它们当前格式合规，但后续如果继续做第三池精修，可以再轻压一次。
- 第四池虽然已经脱离“阅读写生和分装壳链”，但现在仍有 `盘 / 钟 / 历 / 机 / 卷 / 面 / 盆 / 册 / 图` 这些自然家族；后续若继续微调，重点应放在压模板密度，而不是重新引入 `本 / 册 / 板 / 盒 / 篮 / 盘 / 筒 / 卷 / 包`。
- 这轮已经把 `卷 / 席面 / 毯面 / 布面` 压到较合理密度；后续若继续微调，优先关注 `花盆 / 盆栽 / 组合` 的密度平衡，而不是重新把卷物和表面物补回来。
- 当前版本已经可以作为后续同类修改的合格基线，下一步若继续优化，建议优先做“第一池摆盘壳轻压”和“第三池小食饮料单位轻压”，而不是再大改第二池和第四池。
