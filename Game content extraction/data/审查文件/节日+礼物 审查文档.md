# 节日+礼物 审查文档

本文档记录 `节日+礼物` 盲盒物品库的最近一次内容审查结果。修改该类别前必须先读取本文件，并按问题清单逐项处理；本次仅审查并写入文档，不修改 `blind_boxes.py`。

## 总体结论

- `基础数据检查`：四池均为 50 条，池内无精确重复，四池之间无精确字符串重复，未发现字面 `组合` 条目；但第四池存在 `套组`，按规则仍属于组合概念风险。
- `主要问题`：当前内容格式合格，但视觉本体集中在礼物盒、礼品袋、礼篮/礼包/礼套、食品罐桶筒、包装纸/丝带/麻绳卷、蝴蝶结/纸花/装饰球、包装加工工具、夹具、收纳筒篮盒、架座托盘等少数族群；多处只是更换形状、材质、内容物或用途前缀。
- `重点风险`：礼盒/礼袋/包装纸/丝带蝴蝶结同族过密；未发现贺卡、标签、贴纸或价格签条目，但存在节庆符号、蜡烛和封蜡加热类条目，需要避免依赖文字、图案、火焰、发光或细小蜡印。
- `第四池结论`：`scene_expansion_items` 去核心化不足，大量条目是包装加工设备、收纳容器、架座、托盘、垫板、压模盒、切模套组和插盆陶盆；不少更像包装工位、烘焙/茶歇台或装饰维护后场，不像节日礼物画面里自然可放置的中型单物。
- `优先级`：高
- `修改目标`：压缩同视觉本体族，减少容器尾词和工具台后缀；第四池优先改为真实常见、轮廓差异明显、能自然放在节日礼物桌面或礼物陈列边缘的中型辅助单物。

## 修改后复审占位

- `数量是否通过`：待修改后复审。
- `重复是否通过`：待修改后复审；需同时检查精确重复和礼盒、礼袋、礼篮、罐桶筒、卷材、夹具、容器、架座等视觉本体重复。
- `第四池边界是否通过`：待修改后复审；重点确认无 `组合`、无 `套组`、无核心礼物小变体、无成片筒/盒/篮/盆/托盘、架/座/夹模板。
- `场景自然度是否通过`：待修改后复审；重点确认仍是通用节日礼物，不滑向包装加工工位、茶歇烘焙台、咖啡吧台或节庆维护后场。
- `国际化适配是否通过`：待修改后复审；当前无强中式节庆物，但需降低冬季/圣诞符号集中度，避免新增强地域节庆物。
- `找不同可玩性是否通过`：待修改后复审；重点确认没有新增文字标签、小贴纸、价格签、细小夹扣、薄纸片、火焰/发光/反光依赖。
- `剩余风险`：当前最大风险是核心池同族礼物容器过密、第四池工具/容器/架座模板化，以及蜡烛封蜡类的光热和细节依赖；修改阶段应逐项消解，而不是用新的盒、袋、筒、托盘、垫、架、座、夹替换旧问题。
