# 街道+出行 审查文档

本文档记录 `街道+出行` 盲盒物品库最近一轮“审查 -> 修改 -> 复审”的结果。后续若再次调整该类别，仍须先读取本文件，再判断是否需要继续复查和替换。

- `类别`：街道+出行
- `审查日期`：2026-05-07
- `审查范围`：仅 `Game content extraction/data/blind_boxes.py` 中 `街道+出行` 的 `core_items` / `support_items` / `visible_small_items` / `scene_expansion_items`

## 总体结论

- `基础数据检查`：四池均为 50 条，池内无精确重复，四池之间无精确字符串重复，未发现 `组合` 条目。
- `本次修改摘要`：已把核心池从“包袋箱盒 + 票卡证件 + 手机充电配件 + 雨具配件”的拆解式堆叠，改成“少量通勤主体物 + 常见随身物 + 轻量出行物”的混合结构；已把配套池里大量夹扣环片、内页夹、挂环和小附件，改成更厚实的杯套、收纳盒、防雨罩、打气筒、称重器、清洁与卫生辅助物；已把第四池从袋、盒、夹、篮、架、座、盘的取用/保护/临放模板，改成游戏板盘、图册本册、轻设备、计时物、靠垫餐垫和少量前台服务物。
- `第四池结论`：`scene_expansion_items` 已通过中型边界检查，且不再由核心物的收纳壳、保护壳、临放夹和车辆依附支架主导；当前多样性主要来自游戏物、阅读记录物、轻设备、软垫类和少量前台容器，而不是“对象词 + 通用壳体”的连续扩写。
- `场景风险`：未发现品牌名、IP 名称、强中式或强地域物；未发现公交站棚、路牌、长椅、车站闸机等大型街道设施。`滑板`、`折叠滑板车` 等主体物已压缩到少量代表项，且第四池已移除依赖车辆结构的支架托架类写法。
- `优先级`：高
- `修改目标`：本轮已完成；后续若继续精修，优先继续观察 `support_items` 中 `套 / 包 / 带 / 袋` 的密度，以及第四池里 `册 / 本 / 盒 / 器` 是否再次形成单一模板堆叠。
