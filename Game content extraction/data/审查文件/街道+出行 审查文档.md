# 街道+出行 审查文档

## 1. 类别与范围

- 审查对象：`Game content extraction/data/blind_boxes.py` 中 `街道+出行` 的四池内容。
- 本轮结论：当前版本的第一池仍能读出“街道随身出行”的主体场景，但已经混入较多餐具、洗漱、娱乐和日常护理杂件；第二池被 `套 / 袋 / 包 / 盒 / 架 / 绳 / 壳` 的补充壳和保护壳拉散；第四池明显没有通过，已经被写成阅读记录区、涂鸦娱乐区和随身服务设备区的混合池。
- 修改优先级：`scene_expansion_items` > `support_items` > `core_items` > `visible_small_items`。

## 2. 问题总览

- `core_items` 的包袋、雨具、头盔、滑板车、水杯、电源和随身物主场景还在，但“日常杂配件区”和“轻休闲娱乐区”的读感已经开始抬头。
- `support_items` 的主要问题不是与出行无关，而是把真实辅助件、保护壳、收纳壳、补充瓶和护理小壳混在了一池里。
- `visible_small_items` 格式合规，但零食、洗护小瓶、一次性小物和随身小配件成串后，已经很像“通勤出门补给包”。
- `scene_expansion_items` 是当前最大问题：本、册、板、盒、盘、笔筒、画架、相册、收音机和便携设备集中回流，明显偏离“前台自然中型物”的第四池职责。

## 3. 按池问题与必换项

### core_items

- 本池最大问题不是整体失场，而是 `包 / 盒 / 杯 / 本 / 卷 / 管` 壳层和日常娱乐件已经开始挤占主体位。
- 低优先级平衡项：`午餐盒`、`折叠餐杯`、`随身笔记本`、`圆珠笔`、`耳机盒`、`充电线卷`、`充电插头`、`旅行颈枕`、`遮光眼罩`、`随身纸巾包`、`湿巾包`、`拍立得相机`、`便携音箱`、`手持指南针`、`保温饭盒`、`防晒霜管`、`护手霜管`、`彩色臂带`、`折叠坐垫`、`自拍杆`、`折叠餐盘`。
- 这些条目不一定要立刻整片清洗，但它们说明本池已经开始用“随身生活壳、轻娱乐件和护理件”维持密度；后续若要腾位，可优先从这组压缩。
- 当前可作为锚点保留的方向：`随身斜挎包`、`小号双肩包`、`帆布手提袋`、`折叠购物袋`、`软壳登机箱`、`短款钱包`、`卡片夹`、`硬壳太阳镜盒`、`折叠雨伞`、`长柄雨伞`、`轻便雨披`、`防水鞋套`、`自行车头盔`、`滑板`、`折叠滑板车`、`U形自行车锁`、`保温杯`、`运动水瓶`、`外带咖啡杯`、`无线耳机`、`移动电源`、`便携风扇`、`迷你手电筒`、`遮阳帽`、`针织便帽`、`轻薄围巾`、`骑行手套`、`便携零钱包`、`交通卡夹`。

### support_items

- 本池被保护壳、收纳壳和补充壳拉得过散，优先替换：`包带肩垫`、`行李束带`、`拉杆箱轮套`、`背包防雨罩`、`雨伞套`、`雨披收纳袋`、`鞋套收纳袋`、`保温杯套`、`外带杯套`、`餐具收纳盒`、`午餐盒分隔盒`、`耳机保护套`、`充电器收纳包`、`充电线收纳包`、`手机防摔壳`、`折叠杯架`、`折叠餐架`、`交通卡挂绳`、`卡包挂绳`、`钥匙包`、`零钱夹`、`车锁防雨套`、`头盔防雨罩`、`围巾卷带`、`帽子防风绳`、`口罩收纳袋`、`纸巾挂袋`、`湿巾翻盖盒`、`防水证件袋`、`急救药盒`、`彩色臂带套`、`鞋垫包`、`备用鞋带包`、`护手霜包`、`洗手液补充瓶`。
- 本池里也存在“同对象不同补位 + 相同壳体”的伪多样性，优先复查：`水瓶防漏盖`、`水瓶提带`、`手机腕带`、`便携洗手液`、`手部消毒凝胶`。
- 问题根因不是这些词完全不能出现，而是这一池已经同时承担了“真实辅助工具池”“随身保护收纳池”和“日常护理补给池”三种职责。
- 当前更稳的辅助物方向仍应放在真正参与擦拭、防漏、打气、称重、防雨和骑行动作的工具本体，如 `太阳镜擦拭布`、`镜片清洁喷瓶`、`便携打气筒`、`行李称重器`、`便携门锁`、`骑行护膝`、`骑行护肘`、`防晒喷雾瓶`、`雨伞束带`、`衣物除毛刷`。

### visible_small_items

- 整体格式合规，暂不建议整池推倒。
- 低优先级压缩项：`一筒薄荷糖`、`一筒口香糖`、`一排洗手液小瓶`、`一排防晒霜小管`、`一排护手霜小管`、`一盒润唇膏`、`一盒能量棒`、`一包饼干条`、`一袋果干`、`一袋混合坚果`、`一袋薯片`、`一盒巧克力块`、`一排果汁盒`、`一排苏打罐`、`一排迷你水瓶`、`一包纸吸管`、`一包木叉`、`一叠餐巾纸`、`一叠纸盘`、`一卷垃圾袋`、`一卷铝箔纸`、`一盒保冷冰袋`、`一盒茶包`、`一包速溶咖啡`、`一盒热可可粉包`、`一包洗漱小瓶`、`一排咖啡奶球`、`一包小圆镜`、`一盒便签纸`、`一盒折叠衣架`。
- 这些条目本身未必违规，但它们和前两池、第四池的随身补给壳叠加后，会继续放大“街道出行补给与临时生活区”的读感。

### scene_expansion_items

- 图册、本册、相册和记录类回流，必须优先替换：`城市地图册`、`交通图册`、`儿童绘本`、`旅行日记本`、`速写本`、`亲子相册`、`城市相册`、`折页路线图`、`旅行故事本`、`亲子图册`、`观景相册`、`手写留言本`、`城市涂鸦本`。
- 涂鸦写生区器具回流，必须优先替换：`写生画架`、`观景画板`、`绘画笔筒`、`水彩笔筒`、`涂鸦蜡笔筒`。
- 盒、盘、包、桶和托盘家族过密，必须优先替换：`零钱铁盒`、`抽纸盒`、`湿巾盒`、`急救药包`、`防水证件包`、`雨披收纳包`、`点心铁盒`、`木质托盘`、`小号冰桶`、`饮料保温桶`、`彩色臂带盒`。
- 服务设备与临时处理器具回流，必须优先替换：`便携喷雾器`、`保温毯卷`、`便携药盒`、`便携收音机`、`应急收音机`、`便携咖啡机`、`便携录音笔`、`户外驱蚊器`、`喷雾降温器`、`手持计时器`、`便携计时钟`、`手持扩音器`。
- 游戏板家族虽然可见，但与“街道出行”绑定偏弱，建议中高优先级复查：`折叠棋盘`、`拼字游戏板`、`拼图板`、`飞盘游戏盘`、`豆袋投掷板`。
- 当前较接近第四池方向、可作为保留候选的条目相对有限：`旅行小钟`、`电子阅读器`、`花纹餐垫`、`花纹靠垫`。

## 4. 多样性与模板密度

- 第一池当前最明显的压力不是“出行主体物太多”，而是 `包 / 盒 / 杯 / 本 / 卷 / 管` 与日常护理、轻娱乐家族一起抬高了场景杂音。
- 第二池最拥挤的尾词是 `套 / 袋 / 包 / 盒 / 架 / 绳 / 壳`，主要问题是“不同被服务对象 + 相同保护壳/收纳壳”的伪多样性。
- 第三池格式没错，但 `零食小包 / 小瓶 / 一次性小物 / 洗护小件 / 出行小配件` 已经成串，后续只宜轻压，不宜继续扩写。
- 第四池当前最危险的四条串联家族是 `本 / 册 / 相册 / 路线图`、`画架 / 画板 / 笔筒 / 涂鸦本`、`盒 / 包 / 托盘 / 桶`、`收音机 / 录音笔 / 计时器 / 喷雾器 / 服务设备` 这几类出行服务壳体链。

## 5. 第四池专项

- 第四池当前的根因不是个别坏词，而是整池写法站在“路途中阅读记录、涂鸦娱乐和随身服务设备系统”的视角，而不是“街道出行前台场景补充物”的视角。
- 结果是四类问题同时叠加：图册相册和日记本成串、画架画板和笔筒成串、盒包桶盘成串、收音机录音笔和计时设备成串。
- 本类第四池应优先补“普通玩家一眼就能认出的街道路面、等车区、步行出门角、随身停留点”里的自然中型单物，而不是继续补图册、相册、画架、托盘和服务设备。
- 这一池不适合沿现有词表微调；建议修改阶段按高风险家族整段清理，再回填更自然的前台物。

## 6. 修改方向

- 第一池：保留包袋、雨具、头盔、滑板车、锁具、水杯、电源和典型随身出行物这些稳定主体，必要时压缩餐具、洗护、娱乐和护理件的横向平移。
- 第二池：从“保护收纳壳与护理补给壳混池”拉回“真正辅助工具池”，优先保留擦拭、防漏、防雨、称重、打气和骑行防护动作里会直接上手的物件本体。
- 第三池：继续保持 `一包 / 一盒 / 一排 / 一袋 / 一套 / 一叠 / 一卷 / 一筒` 的可见单位格式，但避免零食、小瓶、一次性小物和洗护小件继续膨胀。
- 第四池：按“街道出行前台自然单物 > 场景一眼可读 > 中型边界”重建，禁用图册、相册、画架、笔筒、托盘、铁盒和服务设备作为主要补位手段。
- 收尾时额外检查同后缀家族，尤其是 `本 / 册 / 盒 / 包 / 盘 / 桶 / 筒 / 器` 是否还成片出现。

## 7. 本次修改记录

- 本轮实际修改了 `blind_boxes.py` 中 `街道+出行` 的 `support_items` 和 `scene_expansion_items`，`core_items` 与 `visible_small_items` 暂不大动。
- 第二池的修改方向是把“保护壳、收纳壳、补充瓶和护理小壳混池”拉回“真实擦拭、防漏、防雨、打气、称重和骑行辅助工具池”。

- 第四池本轮按高风险家族整段清理，移除了 `图册 / 相册 / 日记本 / 路线图 / 画架 / 画板 / 笔筒 / 托盘 / 铁盒 / 收纳包 / 计时器 / 收音机类服务设备` 这类阅读记录区、涂鸦娱乐区和随身服务设备区写法。

- 本轮又继续压缩了 `地毯卷 / 毯面 / 席面 / 花盆 / 画册` 这几组高频尾词，把它们都收敛到“每种最多两个”，并用 `路边绿篱盆 / 门店景观盆 / 街角伞桶 / 路口隔离桶 / 单车停靠座 / 雨伞寄放篮 / 待取快递包裹` 这类更贴近街边停靠点的中型物回填。

## 8. 修改后复审

- 当前四池数量均为 50，且池内唯一。
- 第四池与其余三池无重叠。
- 第四池本轮本地复查未命中禁词，且全部通过 `_is_medium_scene_fragment_item` 的中型场景边界校验。
- 第四池中的 `地毯卷 / 毯面 / 席面 / 花盆 / 画册 / 图册` 当前实值分别为 `2 / 2 / 2 / 2 / 2 / 2`，已经不再出现同后缀连续堆叠压读感的问题。
- 这说明本轮已经先把“结构性错误”修掉了，尤其是第四池不再被 `本 / 册 / 盒 / 包 / 盘 / 桶 / 筒 / 器` 或 `毯卷 / 毯面 / 席面 / 花盆 / 画册` 这类串联家族主导。
- 从读感上看，第二池已经明显回到街道出行辅助工具池；第四池也从“阅读记录、涂鸦娱乐和随身服务设备系统”转回“街道出行前台场景补充物”。
- 本节结论仅针对 `街道+出行` 的定点复查，不代表全仓所有类别已全量通过。

## 9. 剩余风险与暂留理由

- 第一池里仍保留了少量随身生活壳、轻娱乐件和护理件，例如 `午餐盒`、`折叠餐杯`、`随身笔记本`、`圆珠笔`、`耳机盒`、`充电线卷`、`充电插头`、`旅行颈枕`、`遮光眼罩`、`随身纸巾包`、`湿巾包`、`拍立得相机`、`便携音箱`、`保温饭盒`、`防晒霜管`、`护手霜管`、`彩色臂带`、`折叠坐垫`、`自拍杆`、`折叠餐盘`；这轮暂不动，是为了先稳住主体池，不把问题从第四池转移到第一池。
- 第三池仍保留了较多零食、小瓶、一次性小物和洗护小件，例如 `一筒薄荷糖`、`一筒口香糖`、`一排洗手液小瓶`、`一排防晒霜小管`、`一排护手霜小管`、`一盒润唇膏`、`一盒能量棒`、`一排果汁盒`、`一排苏打罐`、`一排迷你水瓶`、`一包洗漱小瓶`、`一排咖啡奶球`；它们当前格式合规，但后续如果继续做第三池精修，可以再轻压一次。
- 第四池虽然已经脱离“阅读记录和服务设备壳链”，也压住了 `毯卷 / 毯面 / 席面 / 花盆 / 画册` 的成串问题，但现在仍有 `钟 / 历 / 机 / 盆 / 桶 / 座 / 篮 / 图` 这些自然家族；后续若继续微调，重点应放在继续压模板密度，而不是重新引入 `本 / 册 / 盒 / 包 / 盘 / 桶 / 筒 / 器`。
- 当前版本已经可以作为后续同类修改的合格基线，下一步若继续优化，建议优先做“第一池生活护理杂件轻压”和“第三池零食洗护小件轻压”，而不是再大改第二池和第四池。
