# 沙滩+度假 审查文档

## 1. 类别与范围

- 审查对象：`Game content extraction/data/blind_boxes.py` 中 `沙滩+度假` 的四池内容。
- 本轮结论：当前版本的第一池仍能读出“沙滩度假区”的主体场景，但已经混入较多餐盒、瓶罐、包袋和旅行读物；第二池被 `带 / 绳 / 盒 / 套 / 袋 / 筒 / 篮` 的补充壳和收纳壳拉散；第四池明显没有通过，已经被写成清洁洗脚区、冷饮补给区和整理挂架区的混合池。
- 修改优先级：`scene_expansion_items` > `support_items` > `core_items` > `visible_small_items`。

## 2. 问题总览

- `core_items` 的沙滩巾、遮阳物、泳具、玩沙物和沙滩休闲物主场景还在，但“便携补给包”和“旅行杂配件”的读感已经开始抬头。
- `support_items` 的主要问题不是与沙滩无关，而是把真实辅助件、保护壳、收纳壳、补给壳和绑扎件混在了一池里。
- `visible_small_items` 格式合规，但防晒小样、冷饮配件、零食和迷你玩具成串后，已经很像“沙滩消耗品礼包”。
- `scene_expansion_items` 是当前最大问题：盆、桶、托盘、篮、架、座、整理板、记录本和图册集中回流，明显偏离“前台自然中型物”的第四池职责。

## 3. 按池问题与必换项

### core_items

- 本池最大问题不是整体失场，而是 `包 / 箱 / 桶 / 盒 / 罐 / 瓶 / 册` 壳层和旅行杂物已经开始挤占主体位。
- 低优先级平衡项：`沙滩托特包`、`防水背包`、`干湿分离袋`、`饮料冷藏箱`、`圆形冰桶`、`冷饮随行杯`、`水果餐盒`、`三明治餐盒`、`零食密封罐`、`防晒乳瓶`、`防晒喷雾瓶`、`晒后凝胶瓶`、`旅行读物册`、`防水手机盒`、`冰敷袋`。
- 这些条目不一定要立刻整片清洗，但它们说明本池已经开始用“补给壳、分装壳和旅行配件”维持密度；后续若要腾位，可优先从这组压缩。
- 当前可作为锚点保留的方向：`沙滩巾`、`圆形沙滩毯`、`防潮沙滩垫`、`折叠坐垫`、`充气躺垫`、`遮阳伞`、`宽檐遮阳帽`、`草编太阳帽`、`太阳镜`、`便携风扇`、`凉感颈枕`、`保温水壶`、`降温毛巾`、`防晒披肩`、`沙滩拖鞋`、`防滑水鞋`、`泳镜`、`浮潜面罩`、`泳帽`、`脚蹼`、`浮板`、`游泳圈`、`充气臂圈`、`潜水呼吸管`、`浮水玩具`、`沙滩球`、`飞盘`、`沙滩拍`、`玩沙桶`、`玩沙筛`、`沙堡模具桶`、`沙滩风筝`。

### support_items

- 本池被绑扎件、保护壳和收纳壳拉得过散，优先替换：`沙滩巾绑带`、`浴巾收纳带`、`防风伞绳`、`伞绳绕线器`、`伞座沙袋`、`太阳镜硬盒`、`保冷剂盒`、`冷藏包肩带`、`点心分食盘`、`旅行分装瓶`、`泳镜硬壳盒`、`泳帽收纳筒`、`水鞋沥水网篮`、`脚蹼束带`、`浮板腕带`、`降温毛巾筒`、`湿衣袋束口扣`、`泳具收纳袋`、`沙滩拍套`、`镜片防雾布盒`。
- 本池也存在“同动作不同对象 + 相同壳体”的伪多样性，优先复查：`遮阳帽绳`、`沙滩垫束带`、`游泳圈修补包`、`冷藏箱冰板`、`餐盒保冷垫`。
- 问题根因不是这些词完全不能出现，而是这一池已经同时承担了“真实小工具池”“绑扎保护件池”和“海边整理收纳池”三种职责。
- 当前更稳的辅助物方向仍应放在真正参与固定、擦拭、开瓶、充气、玩沙、防雾和清洁动作的工具本体，如 `大号毛巾夹`、`垫角固定钉`、`沙滩伞地钉`、`眼镜擦拭布`、`水杯硅胶套`、`随行杯盖`、`杯身防滑圈`、`冰桶夹钳`、`折叠小餐叉`、`零食封口夹`、`饮料开瓶器`、`防晒涂抹海绵`、`防水梳`、`帽檐清洁刷`、`泳镜防雾瓶`、`手动充气泵`、`玩沙铲`、`沙滩耙`、`沙模杯`、`玩沙清洗刷`、`相机浮力腕绳`、`风扇充电底座`、`沙滩鞋清洁刷`、`防晒袖套`。

### visible_small_items

- 整体格式合规，暂不建议整池推倒。
- 低优先级压缩项：`一排防晒小样瓶`、`一排晒后凝胶小瓶`、`一盒芦荟胶小管`、`一盒润唇膏管`、`一盒冰镇贴`、`一盒泳镜防雾布`、`一排泳镜防雾液`、`一盒毛巾夹`、`一捆沙滩巾绑带`、`一包沙滩伞地钉`、`一卷防风伞绳`、`一盒保冷剂`、`一包小冰袋`、`一筒饮料搅拌棒`、`一盒便携游戏牌`、`一盒沙滩扑克牌`、`一盒沙滩棋子`、`一袋爆米花`、`一袋薯片`、`一盒水果软糖`、`一筒防水纸杯`、`一盒冷饮杯套`。
- 这些条目本身未必违规，但它们与前两池、第四池的补给和分装壳叠加后，会继续放大“沙滩补给与零食耗材区”的读感。

### scene_expansion_items

- 清洁、洗脚和临时处理物回流，必须优先替换：`冲脚盆`、`折叠洗脚盆`、`清水桶`、`沙滩清洁盆`、`冲脚喷雾器`、`脚底清洁刷`、`防沙鞋刷`。
- 盒、托盘、提桶、保冷筒和分装桶回流，必须优先替换：`纸巾木盒`、`湿巾盒`、`餐具盒`、`水果托盘`、`点心托盘`、`冷饮托盘`、`冷饮提桶`、`零食桶`、`保冷筒`、`清洁托盘`、`饮料分装桶`、`餐巾盒`。
- 篮、架、座和整理板家族过密，必须优先替换：`毛巾篮`、`玩具篮`、`湿衣篮`、`出门杂物篮`、`遮阳帽架`、`毛巾挂架`、`水鞋晾架`、`沙滩拍座`、`飞盘座`、`便携镜架`、`饮料插地座`、`太阳镜展示板`、`浮具整理板`、`泳具整理板`、`泳圈整理板`、`玩沙工具座`。
- 记录本、图册和提示板家族回流，必须优先替换：`游玩记录本`、`海边读物册`、`旅行清单板`、`防晒提示板`、`沙滩图册`、`休闲读物册`、`游玩打卡本`、`海边留言本`。
- 当前较接近第四池方向、可作为保留候选的条目相对有限：`折叠垃圾桶`、`便携收音机`、`沙滩小钟`、`沙漏计时器`。

## 4. 多样性与模板密度

- 第一池当前最明显的压力不是“玩水物太多”，而是 `包 / 箱 / 桶 / 盒 / 罐 / 瓶 / 册` 与补给餐食家族一起抬高了场景杂音。
- 第二池最拥挤的尾词是 `带 / 绳 / 盒 / 套 / 袋 / 筒`，主要问题是“不同被服务对象 + 相同保护壳/收纳壳”的伪多样性。
- 第三池格式没错，但 `防晒小瓶 / 零食小包 / 小餐具 / 迷你玩具 / 冷饮配件` 已经成串，后续只宜轻压，不宜继续扩写。
- 第四池当前最危险的四条串联家族是 `盆 / 桶 / 刷 / 喷雾器`、`托盘 / 盒 / 提桶 / 保冷筒 / 分装桶`、`篮 / 架 / 座 / 整理板`、`记录本 / 图册 / 提示板 / 留言本` 这几类海边服务壳体链。

## 5. 第四池专项

- 第四池当前的根因不是个别坏词，而是整池写法站在“海边补给、清洁和整理系统”的视角，而不是“沙滩前台场景补充物”的视角。
- 结果是四类问题同时叠加：洗脚清洁物成串、餐食冷饮分装壳成串、整理架板成串、记录本和图册成串。
- 本类第四池应优先补“普通玩家一眼就能认出的沙滩巾区、遮阳区、玩沙区、休闲坐躺区”里的自然中型单物，而不是继续补洗脚盆、整理板、挂架、托盘和打卡本。
- 这一池不适合沿现有词表微调；建议修改阶段按高风险家族整段清理，再回填更自然的前台物。

## 6. 修改方向

- 第一池：保留沙滩巾、遮阳物、泳具、玩沙物和休闲主体，必要时压缩包袋、餐盒、瓶罐、分装壳和旅行读物的横向平移。
- 第二池：从“绑扎件与收纳壳混池”拉回“真正辅助工具池”，优先保留固定、擦拭、开瓶、充气、防雾、清洁和玩沙动作里会直接上手的物件本体。
- 第三池：继续保持 `一盒 / 一包 / 一排 / 一套 / 一筒 / 一卷 / 一袋` 的可见单位格式，但避免防晒小瓶、零食包、冷饮配件和迷你玩具继续膨胀。
- 第四池：按“沙滩前台自然单物 > 场景一眼可读 > 中型边界”重建，禁用洗脚盆、清洁盆、托盘、提桶、整理板、挂架和记录本作为主要补位手段。
- 收尾时额外检查同后缀家族，尤其是 `盆 / 桶 / 盒 / 托盘 / 篮 / 架 / 座 / 板 / 本 / 册` 是否还成片出现。

- 第二池的修改方向是把“绑带、收纳壳、保护壳、整理件混池”拉回“真实固定、擦拭、充气、防雾、玩沙和沙滩清洁工具池”。

## 8. 修改后复审

- 当前四池数量均为 50，且池内唯一。
- 第四池与其余三池无重叠。
- 第四池本轮本地复查未命中禁词，且全部通过 `_is_medium_scene_fragment_item` 的中型场景边界校验。
- 这说明本轮已经先把“结构性错误”修掉了，尤其是第四池不再被 `盆 / 桶 / 托盘 / 篮 / 架 / 座 / 板 / 本 / 册` 这类海边服务壳主导。
- 从读感上看，第二池已经明显回到沙滩工具池；第四池也从“海边补给、清洁和整理系统”转回“沙滩前台场景补充物”。
- 进一步按后缀统计复查后，第四池里的 `图册 / 画册 / 席面 / 布面 / 毯面` 已分别压到 2 条；`盆类` 也同步压到 2 条，满足这轮补充要求。
- 本节结论仅针对 `沙滩+度假` 的定点复查，不代表全仓所有类别已全量通过。

## 9. 剩余风险与暂留理由

- 第一池里仍保留了少量补给壳和旅行杂配件，例如 `沙滩托特包`、`防水背包`、`干湿分离袋`、`饮料冷藏箱`、`圆形冰桶`、`水果餐盒`、`三明治餐盒`、`零食密封罐`、`防晒乳瓶`、`防晒喷雾瓶`、`晒后凝胶瓶`、`旅行读物册`、`防水手机盒`、`冰敷袋`；这轮暂不动，是为了先稳住主体池，不把问题从第四池转移到第一池。
- 第三池仍保留了一些防晒小瓶、冷饮配件和零食单位，例如 `一排防晒小样瓶`、`一排晒后凝胶小瓶`、`一盒保冷剂`、`一包小冰袋`、`一筒饮料搅拌棒`、`一袋爆米花`、`一袋薯片`；它们当前格式合规，但后续如果继续做第三池精修，可以再轻压一次。
- 第四池虽然已经脱离“补给壳链”，但现在仍有 `钟 / 历 / 机 / 卷 / 面 / 盆 / 册 / 图` 这些自然家族；后续若继续微调，重点应放在压模板密度，而不是重新引入 `盆 / 桶 / 盒 / 托盘 / 篮 / 架 / 座 / 板 / 本`。
- 这轮已经把 `图册 / 画册 / 席面 / 布面 / 毯面 / 盆类` 压到较合理密度；后续若继续微调，优先关注 `卷 / 组合 / 机` 的密度平衡，而不是重新把读物、表面物和盆类补回来。
- 当前版本已经可以作为后续同类修改的合格基线，下一步若继续优化，建议优先做“第一池补给壳轻压”和“第三池防晒零食配件轻压”，而不是再大改第二池和第四池。
