# 营地+露营 审查文档

## 1. 类别与范围

- 审查对象：`Game content extraction/data/blind_boxes.py` 中 `营地+露营` 的四池内容。
- 本轮结论：当前版本的第一池仍能读出“营地露营区”的主体场景，但已经混入较多棋盒、日志本、相机、音箱和日常穿戴配件；第二池被 `盆 / 桶 / 篮 / 盒 / 盘 / 包 / 架` 的补充壳和清洁壳拉散；第四池明显没有通过，已经被写成观察记录区、写生娱乐区和服务设备区的混合池。
- 修改优先级：`scene_expansion_items` > `support_items` > `core_items` > `visible_small_items`。

## 2. 问题总览

- `core_items` 的帐篷、露营椅、桌子、睡袋、炉具和照明主场景还在，但“营地生活杂配件”和“轻娱乐记录物”的读感已经开始抬头。
- `support_items` 的主要问题不是与露营无关，而是把真实辅助工具、清洁洗漱物、收纳壳、餐厨分装壳和后场服务物混在了一池里。
- `visible_small_items` 格式合规，但零食、小食、洗护耗材和娱乐文具成串后，已经很像“营地便利补给包”。
- `scene_expansion_items` 是当前最大问题：本、册、板、盘、桶、盒、笔筒、画架、显微镜和记录器集中回流，明显偏离“前台自然中型物”的第四池职责。

## 3. 按池问题与必换项

### core_items

- 本池最大问题不是整体失场，而是 `盒 / 本 / 包 / 杯 / 卷` 壳层和娱乐记录物已经开始挤占主体位。
- 低优先级平衡项：`徒步背包`、`露营收纳箱`、`硬壳保冷箱`、`搪瓷餐盘`、`搪瓷杯`、`折叠棋盘`、`急救包`、`洗漱包`、`毛巾卷`、`双筒望远镜`、`便携音箱`、`拍立得相机`、`扑克牌盒`、`飞盘`、`泡泡机`、`风筝线盘`、`折叠雨伞`、`露营拖鞋`、`针织便帽`、`围巾`、`户外坐垫`、`保温咖啡杯`、`双陆棋盒`、`营地日志本`、`便携对讲机`。
- 这些条目不一定要立刻整片清洗，但它们说明本池已经开始用“生活杂件、娱乐盒体和记录物”维持密度；后续若要腾位，可优先从这组压缩。
- 当前可作为锚点保留的方向：`圆顶帐篷`、`折叠露营椅`、`蛋卷桌`、`睡袋卷`、`防潮垫`、`露营毯`、`充气靠枕`、`保温水壶`、`运动水瓶`、`露营提灯`、`便携炉具`、`烧水壶`、`咖啡壶`、`手摇磨豆机`、`平底煎锅`、`烤网`、`保暖斗篷`、`防虫喷雾瓶`、`防晒霜管`、`雨披`、`手持指南针`、`营地灯`、`多功能工具钳`、`橡胶营地锤`、`折叠手锯`。

### support_items

- 本池被清洁壳、收纳壳和洗漱壳拉得过散，优先替换：`帐篷修补包`、`折叠水盆`、`折叠洗手盆`、`便携水桶`、`餐具沥水篮`、`咖啡滤纸包`、`调味瓶组`、`密封餐盒`、`清洁海绵包`、`擦手布卷`、`折叠垃圾桶`、`野餐垃圾袋`、`抽纸包`、`湿巾包`、`便携洗手液`、`手部消毒凝胶`、`便携药箱`、`折叠储物篮`、`风筝收线盘`、`露营杯架`、`保温毯卷`、`餐盘沥水架`、`折叠洗菜盆`、`小号收纳盘`、`调味收纳盒`、`桌面调味架`、`肥皂沥水盘`、`手提洗漱杯`。
- 本池里也存在“同动作不同对象 + 相同壳体”的伪多样性，优先复查：`手压打气泵`、`脚踩打气泵`、`保冷冰袋`、`保温壶刷`、`洗碗刷`。
- 问题根因不是这些词完全不能出现，而是这一池已经同时承担了“真实辅助工具池”“营地洗漱清洁池”和“餐厨收纳分装池”三种职责。
- 当前更稳的辅助物方向仍应放在真正参与搭建、烹饪、取水、开瓶、夹取、挡风和简单清洁动作的工具本体，如 `防水地布`、`炉具挡风板`、`烧烤夹`、`锅盖夹`、`锅铲`、`长柄汤勺`、`咖啡滤杯`、`开瓶器`、`罐头开启器`、`隔热手套`、`余烬铲`、`靴子刷`、`小号扫刷`、`镜头清洁刷`、`露营围裙`。

### visible_small_items

- 整体格式合规，暂不建议整池推倒。
- 低优先级压缩项：`一盒燕麦棒`、`一袋棉花糖`、`一袋椒盐脆饼`、`一袋薯片`、`一袋爆米花`、`一包曲奇饼干`、`一包夹心饼干`、`一盒巧克力块`、`一包牛肉干`、`一包果干`、`一袋混合坚果`、`一盒茶包`、`一包速溶咖啡`、`一盒热可可粉包`、`一排果汁盒`、`一排苏打罐`、`一排迷你水瓶`、`一盒调味小罐`、`一盒创可贴`、`一卷纱布绷带`、`一包酒精棉片`、`一排洗手液小瓶`、`一排防晒霜小管`、`一排驱蚊喷雾小瓶`、`一盒扑克牌`、`一盒多米诺骨牌`、`一包彩色粉笔`、`一筒泡泡液`、`一盒蜡笔`、`一盒彩色铅笔`、`一盒保冷冰袋`。
- 这些条目本身未必违规，但它们和前两池、第四池的补给、娱乐和护理壳叠加后，会继续放大“营地便利补给区”的读感。

### scene_expansion_items

- 故事本、图册、图鉴和记录板家族回流，必须优先替换：`露营故事本`、`观星图册`、`植物图鉴`、`观鸟图册`、`花草拼贴板`、`露营留言本`、`速写本`、`折页路线图`、`营地记录板`。
- 观察记录器具和服务设备回流，必须优先替换：`便携显微镜`、`便携录音笔`、`儿童显微镜`、`电子记分器`、`手持记分器`、`便携咖啡机`、`便携净水器`、`便携打气机`、`太阳能充电器`。
- 盘、桶、盒和托盘家族过密，必须优先替换：`急救药包`、`抽纸盒`、`湿巾盒`、`点心铁盒`、`木质托盘`、`搪瓷水桶`、`小号冰桶`。
- 写生娱乐区器具过密，必须优先替换：`水彩颜料盘`、`户外画架`、`写生画板`、`绘画笔筒`、`涂鸦蜡笔筒`、`风筝卷轴`。
- 后场服务物与近义重复项混入，必须整体压缩：`户外驱蚊器`、`喷雾降温器`、`对讲机`、`羊毛毯卷`、`花纹靠垫`、`野餐席面`、`户外餐垫`、`防水证件包`、`雨披收纳包`。
- 当前较接近第四池方向、可作为保留候选的条目相对有限：`拼字游戏板`、`磁性飞镖盘`、`飞盘游戏盘`、`豆袋投掷板`、`手摇收音机`、`应急收音机`、`营地小钟`。

## 4. 多样性与模板密度

- 第一池当前最明显的压力不是“露营主体物太多”，而是 `盒 / 本 / 包 / 卷 / 杯` 与娱乐记录家族一起抬高了场景杂音。
- 第二池最拥挤的尾词是 `盆 / 桶 / 篮 / 盒 / 盘 / 包 / 架`，主要问题是“不同被服务对象 + 相同清洁壳/收纳壳”的伪多样性。
- 第三池格式没错，但 `零食小包 / 小饮料 / 护理耗材 / 小游戏 / 小文具` 已经成串，后续只宜轻压，不宜继续扩写。
- 第四池当前最危险的四条串联家族是 `本 / 册 / 图鉴 / 记录板`、`托盘 / 桶 / 盒 / 药包`、`画架 / 画板 / 笔筒 / 颜料盘`、`显微镜 / 录音笔 / 计分器 / 充电器 / 设备器` 这几类营地服务壳体链。

## 5. 第四池专项

- 第四池当前的根因不是个别坏词，而是整池写法站在“营地观察记录、写生娱乐和服务设备系统”的视角，而不是“营地前台场景补充物”的视角。
- 结果是四类问题同时叠加：故事本和图册成串、托盘水桶和盒体成串、写生绘画器具成串、显微镜录音笔和充电设备成串。
- 本类第四池应优先补“普通玩家一眼就能认出的帐篷区、桌椅区、火炉区、休闲互动区”里的自然中型单物，而不是继续补图册、记录板、画架、笔筒、净水器和打气机。
- 这一池不适合沿现有词表微调；建议修改阶段按高风险家族整段清理，再回填更自然的前台物。

## 6. 修改方向

- 第一池：保留帐篷、桌椅、睡眠、防潮、照明、炉具和基础户外工具这些稳定主体，必要时压缩棋盒、日志本、相机、音箱和日常穿戴配件的横向平移。
- 第二池：从“洗漱清洁壳与餐厨收纳壳混池”拉回“真正辅助工具池”，优先保留搭建、烹饪、夹取、挡风、开瓶、取水和简单清洁动作里会直接上手的物件本体。
- 第三池：继续保持 `一盒 / 一袋 / 一包 / 一排 / 一套 / 一卷 / 一筒` 的可见单位格式，但避免零食、护理耗材、小游戏和小文具继续膨胀。
- 第四池：按“营地前台自然单物 > 场景一眼可读 > 中型边界”重建，禁用图册、本册、记录板、托盘、水桶、画架、笔筒和服务设备作为主要补位手段。
- 收尾时额外检查同后缀家族，尤其是 `本 / 册 / 板 / 盒 / 桶 / 盘 / 包 / 筒 / 器` 是否还成片出现。

## 8. 修改后复审

- 当前四池数量均为 50，且池内唯一。
- 第四池与其余三池无重叠。
- 第四池本轮本地复查未命中禁词，且全部通过 `_is_medium_scene_fragment_item` 的中型场景边界校验。
- 这说明本轮已经先把“结构性错误”修掉了，尤其是第四池不再被 `本 / 册 / 板 / 盒 / 桶 / 盘 / 包 / 筒 / 器` 这类观察记录和服务设备壳主导。
- 从读感上看，第二池已经明显回到露营辅助工具池；第四池也从“观察记录、写生娱乐和服务设备系统”转回“营地前台场景补充物”。
- 进一步按后缀统计复查后，第四池里的 `毯面 / 席面` 已分别压到 2 条，满足这一轮补充要求。
- 本轮继续复查后，第四池里以 `卷` 结尾的条目也已压到 `2` 条，当前仅保留 `营地地毯卷 / 防水地毯卷`。
- 当前再次执行 `test_blind_box_content_model.py` 时，全量 14 项规则测试已通过；本类本轮微调没有引入新的共享规则回归。
- 本节结论仅针对 `营地+露营` 的定点复查，不代表全仓所有类别已全量通过。

## 9. 剩余风险与暂留理由

- 第一池里仍保留了少量生活杂件、娱乐盒体和记录物，例如 `徒步背包`、`露营收纳箱`、`硬壳保冷箱`、`搪瓷餐盘`、`搪瓷杯`、`折叠棋盘`、`急救包`、`洗漱包`、`毛巾卷`、`双筒望远镜`、`便携音箱`、`拍立得相机`、`扑克牌盒`、`双陆棋盒`、`营地日志本`、`便携对讲机`；这轮暂不动，是为了先稳住主体池，不把问题从第四池转移到第一池。
- 第三池仍保留了较多零食、小饮料、护理耗材和小游戏单位，例如 `一盒燕麦棒`、`一袋棉花糖`、`一袋薯片`、`一排果汁盒`、`一排苏打罐`、`一排迷你水瓶`、`一盒创可贴`、`一卷纱布绷带`、`一排洗手液小瓶`、`一排防晒霜小管`、`一排驱蚊喷雾小瓶`、`一盒扑克牌`、`一盒多米诺骨牌`、`一筒泡泡液`；它们当前格式合规，但后续如果继续做第三池精修，可以再轻压一次。
- 第四池虽然已经脱离“观察记录和服务设备壳链”，但现在仍有 `盘 / 钟 / 历 / 机 / 卷 / 面 / 盆 / 册 / 图` 这些自然家族；后续若继续微调，重点应放在压模板密度，而不是重新引入 `本 / 册 / 板 / 盒 / 桶 / 盘 / 包 / 筒 / 器`。
- 这轮已经把 `毯面 / 席面 / 卷` 压到较合理密度；后续若继续微调，优先关注 `盆类 / 图册` 的密度平衡，而不是重新把卷物或表面物补回来。
- 当前版本已经可以作为后续同类修改的合格基线，下一步若继续优化，建议优先做“第一池生活杂件轻压”和“第三池零食护理娱乐单位轻压”，而不是再大改第二池和第四池。
