## 总体结论

状态：`已修改并复审`

这类当前的主要风险，不是场景跑偏得太远，而是容易在正确场景里继续用模板凑数：

- `core_items` 天然会有瓶、罐、盒、管密度，但如果继续只靠护肤和彩妆容器横向扩写，就会让差异只剩内容物说明。
- `support_items` 容易滑成一串 `小工具 + 套/盒`，看起来像修容工具清单，而不是有轮廓差异的辅助件集合。
- `visible_small_items` 容易回到 `一盒 / 一排 / 一套` 反复套在发饰、彩妆笔膏和小工具上，导致模板感过强。
- `scene_expansion_items` 最容易掉回 `包 / 袋 / 托盘 / 相册 / 电子小设备` 的单一路径，看似贴场景，实则多样性不足。

## 本次修改方向

- `core_items`：保留卧室梳妆最核心的镜、发型工具、彩妆、护肤、香氛、首饰和软包主线，同时补入 `指甲油瓶`、`假睫毛盒`、`美容滚轮` 这类更能拉开轮廓的主体物。
- `support_items`：压缩 `套` 链，把一部分条目改为更自然的 `盒 / 垫 / 筒 / 包 / 杯 / 耙 / 刷` 等可识别辅助物，减少纯工具说明书感。
- `visible_small_items`：打散 `一排 + 细长管笔膏` 的旧写法，改成更分散的 `一盒 / 一盘 / 一筒 / 一套 / 一束 / 一卷 / 一叠` 混合单位。
- `scene_expansion_items`：保留少量真实常见的小设备，但把第四池拉回到纸品、台历、香薰、花盆、托盘、相册、卷包、挂架、浅篮、软垫这些更自然的前台中型物，减少纯包袋和平铺托盘的单线扩写。

## 本次主要压掉的伪多样性

- `不同修容工具 + 相同套/盒壳`
- `不同彩妆笔膏 + 相同一排模板`
- `不同前台杂物 + 相同包袋壳体`
- `不同陈设对象 + 相同托盘/相册壳体`
- `同场景正确但轮廓未拉开` 的伪多样性

## 复审结果

- 四池均为 50 条，池内唯一。
- `scene_expansion_items` 与其他三池无精确重复。
- `scene_expansion_items` 已通过中型场景碎片边界自检，未命中超大物和微小信息面禁区。
- 当前版本已明显压低 `support_items` 的 `套` 链、`visible_small_items` 的 `一排/一盒` 链，以及第四池的 `包/袋/托盘/相册` 单一路径。
- 第四池现在更像卧室梳妆前台和床边个人整理物，而不是工具池、托盘池或床头电子杂物堆。

## 保留风险

- `卧室+梳妆` 天然仍会保留较高的瓶罐盒管密度，这是类别识别所必需；后续补写时要继续防止只在护肤和彩妆容器里做小变体。
- `support_items` 仍然会有较多盒类，因为修容和旅行辅助件确实常见盒装；但后续若继续补写，优先补轮廓不同的垫、杯、耙、刷和软包，不要继续堆小工具盒。
- `scene_expansion_items` 目前仍有少量卷包、礼袋、托盘和相册，这是可接受的前台物；但后续若继续补写，不能再滑回“连续包袋”或“连续托盘”的壳体路线。
