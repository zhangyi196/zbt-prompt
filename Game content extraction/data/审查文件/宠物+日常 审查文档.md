# 宠物+日常 审查文档

## 1. 类别与范围

- 审查对象：`Game content extraction/data/blind_boxes.py` 中 `宠物+日常` 的四池内容。
- 本轮结论：当前版本的第一池仍在场景内，但已经明显混入较多容器壳和外出装备壳；第二池被护理配件、连接带、护套、收纳筒和清洗辅件拉得过散；第四池仍然没有通过，核心问题是它被写成了宠物备品间、清洁区和收纳区的混合池。
- 修改优先级：`scene_expansion_items` > `support_items` > `core_items` > `visible_small_items`。

## 2. 问题总览

- `core_items` 的宠物角和地面场景感还在，但 `桶 / 罐 / 盒 / 包 / 箱 / 器` 家族已经偏密，部分条目更像用品壳或运输壳横向平移。
- `support_items` 的主要问题不是不属于宠物，而是把真正的护理工具、连接配件、保护壳、收纳壳和清洗辅件混在了一池里。
- `visible_small_items` 格式合规，但旅行洗护件、清洁耗材、外出挂扣和服饰配件已经开始放大“宠物补给区”的读感。
- `scene_expansion_items` 是当前最大问题：盒、包、篮、桶、架、托盘、清洁盆和家用电器同时回流，已经偏离“宠物前台自然单物”的第四池职责。

## 3. 按池问题与必换项

### core_items

- 本池没有出现必须整片清洗的问题，但容器壳、运输壳和用品壳已经过密。
- 低优先级平衡项：`粮食密封桶`、`零食罐`、`尿垫托盘`、`宠物湿巾盒`、`洗爪杯`、`外出背包`、`便携航空箱`、`宠物帐篷`、`滚球盘`。
- 这些条目不是立刻必换，而是说明本池已经在用“用品本体 + 容器壳”或“运输壳 + 外出壳”维持密度；后续若要腾位，可优先从这组下手。
- 当前可作为锚点保留的方向：`自动喂食器`、`高脚食盆`、`双碗食盆`、`慢食碗`、`防溢水碗`、`自动饮水器`、`猫砂盆`、`封闭猫砂盆`、`宠物香波瓶`、`宠物毛巾`、`梳毛刷`、`指甲剪`、`防舔项圈`、`宠物窝`、`圆形睡垫`、`宠物毯`、`牵引绳`、`胸背带`、`项圈`、`宠物雨衣`、`救生衣`、`橡胶球`、`绳结球`、`漏食球`、`猫抓板`、`逗猫棒`、`抓柱`、`嗅闻垫`、`咬胶骨`、`磨牙棒`、`飞盘`、`浮水球`、`猫薄荷娃娃`、`发声球`、`宠物枕`。

### support_items

- 保护壳、连接件和收纳壳过密，优先替换：`罐头保鲜盖`、`食盆防滑垫`、`食盆托垫`、`拾便袋挂筒`、`湿巾抽取盖`、`耳部清洁棉包`、`眼部清洁棉包`、`护理指套盒`、`梳毛梳套`、`梳毛围布`、`护理围裙`、`外出包垫`、`安全带连接带`、`牵引延长带`、`牵引手柄套`、`胸背护垫`、`项圈护套`、`雨衣束带`、`毛巾束带`、`响片腕带`、`零食袋腰夹`、`玩具清洗网篮`、`玩具收纳篮`、`飞盘收纳筒`、`玩具绳收纳筒`、`训练标志盘`、`奖励球盒`、`护爪蜡盒`、`宠物鞋套盒`。
- 清洁和护理工具整体可保留较多，但仍要避免继续往“护理配件箱”方向扩写。
- 当前更稳的辅助物方向仍应放在真正参与喂食、清洁、洗澡、梳毛、口腔护理和训练动作的工具本体，如 `粮铲`、`量粮杯`、`喂食勺`、`罐头勺`、`开罐器`、`饮水器清洗刷`、`水瓶清洁刷`、`猫砂筛铲`、`拾便夹`、`洗澡按摩刷`、`起泡量杯`、`软毛牙刷`、`宠物牙膏`、`指甲锉`、`毛发滚筒`、`除毛刮板`、`吹毛梳`、`公仔清洁刷`、`磨牙清洁刷`。

### visible_small_items

- 整体格式合规，暂不建议整池推倒。
- 低优先级压缩项：`一包香波旅行瓶`、`一排洗护小瓶`、`一盒指套牙刷`、`一盒训练响片`、`一套外出折叠碗`、`一包清洁手套`、`一盒小领结`、`一排宠物围巾`、`一盒胸背挂扣`、`一袋雨衣扣带`。
- 这些条目本身不违规，但它们和前两池、第四池的配件壳条目叠加后，会进一步放大“宠物补给包”和“外出附件区”的读感。

### scene_expansion_items

- 纸品壳、收纳壳和提包回流，必须优先替换：`抽纸盒`、`湿巾包`、`拾便袋盒`、`香波包`、`外出提包`、`玩具提篮`、`球类篮`、`公仔篮`、`零食桶`、`储粮袋`、`猫砂桶`、`奖励盒`、`宠物用品包`、`零食提篮`、`宠物衣物篮`。
- 架、筒和托盘家族回流，必须优先替换：`雨衣挂架`、`毛毯架`、`毛巾架`、`玩具架`、`项圈挂架`、`项圈筒`、`磨牙棒筒`、`逗猫棒筒`、`牵引绳架`、`食盆托盘`、`清洁托盘`、`零食盘`、`饮水托盘`、`围裙挂钩座`。
- 后场清洁和清洗物回流，必须优先替换：`清洁桶`、`洗爪盆`、`泡澡盆`、`玩具清洗盆`、`梳毛清洗盆`、`毛发回收桶`。
- 核心物近义变体和家电混入，必须优先替换：`宠物饮水机`、`自动喂食机`、`加湿器`、`暖风机`、`烘干机`、`空气净化机`、`吸毛器`、`训练记录板`。
- 当前较接近第四池方向、可作为保留候选的仅有少数：`桌面小钟`、`沙漏计时器`、`翻页台历`、`桌面月历`、`门口擦爪垫`、`猫砂脚垫`。

## 4. 多样性与模板密度

- 第一池最拥挤的家族是 `桶 / 罐 / 盒 / 包 / 箱 / 器`，说明主体池已经出现用品壳、运输壳和容器壳横向平移。
- 第二池最拥挤的尾词是 `套 / 带 / 垫 / 护 / 筒 / 篮 / 盒`，主要问题是“不同宠物对象 + 相同护理配件壳”的伪多样性。
- 第三池格式没错，但 `旅行装 / 洗护件 / 外出挂扣 / 饰品配件` 已经成串，后续只宜轻压，不宜继续扩写。
- 第四池当前最危险的四条串联家族是 `盒 / 包 / 提包 / 篮 / 桶`、`架 / 筒 / 托盘 / 挂架`、`清洗盆 / 清洁桶 / 回收桶`、`家电和训练外设` 这类备品区壳体链。

## 5. 第四池专项

- 第四池当前的根因不是单个坏词，而是整池写法站在“宠物备品管理和清洁流程”视角，而不是“宠物角前台场景”视角。
- 结果是四类问题同时叠加：收纳桶篮和提包堆叠、挂架筒托盘堆叠、清洗盆桶堆叠、家电和训练外设混入。
- 本类第四池应优先补“普通玩家一眼就能认出的宠物角、地垫区、门边用品区前台中型物、自然外露的稳定场景补充物”，而不是继续补盒、包、桶、架、筒、托盘和清洁盆。
- 这一池不适合沿现有词表微调；建议修改阶段按高风险家族整段清理，再回填更自然的前台物。

## 6. 修改方向

- 第一池：保留稳定的喂食物、睡垫物、出行穿戴物和宠物玩具体系，必要时压缩运输壳、容器壳和用品壳的横向平移。
- 第二池：从“护理配件和连接件混池”拉回“真正辅助工具池”，优先保留能直接参与喂食、清洁、洗澡、梳毛、护理和训练动作的工具本体。
- 第三池：继续保持 `一包 / 一盒 / 一筒 / 一排 / 一套 / 一袋` 的可见单位格式，但避免旅行装、挂扣配件和装饰配件继续膨胀。
- 第四池：按“宠物角前台自然单物 > 场景一眼可读 > 中型边界”重建，禁用提包、桶篮、挂架、托盘和清洁流程物作为主要补位手段。
- 收尾时额外检查同后缀家族，尤其是 `盒 / 包 / 桶 / 篮 / 架 / 筒 / 托盘 / 盆` 是否还成片出现。

## 7. 本次修改记录

- 本轮实际修改了 `blind_boxes.py` 中 `宠物+日常` 的 `support_items` 和 `scene_expansion_items`，`core_items` 与 `visible_small_items` 暂不大动。
- 第二池的修改方向是把“护套、连接带、收纳筒、挂件壳、清洗辅件混池”拉回“真实喂食、清洁、洗护、梳毛、口腔护理、训练工具池”。
- 具体清掉了 `保鲜盖 / 防滑垫 / 托垫 / 挂筒 / 抽取盖 / 棉包 / 指套盒 / 围布 / 围裙 / 包垫 / 连接带 / 延长带 / 手柄套 / 护垫 / 护套 / 束带 / 腕带 / 腰夹 / 网篮 / 收纳篮 / 收纳筒 / 标志盘 / 奖励球盒 / 护爪蜡盒 / 鞋套盒` 这类弱本体配件壳。
- 回填时改用 `喂药器 / 喂药滴管 / 饮水嘴疏通刷 / 猫砂平整铲 / 拾便铲 / 除味喷瓶 / 洗耳瓶 / 洗眼滴管 / 耳毛镊子 / 排梳 / 开结梳 / 底毛耙 / 圆头修毛剪 / 训练响笛 / 目标指示棒` 等真实工具本体。
- 第四池本轮按高风险家族整段清理，移除了 `盒 / 包 / 提篮 / 桶 / 架 / 筒 / 托盘 / 清洗盆 / 回收桶 / 训练记录板 / 宠物饮水机 / 自动喂食机` 这类备品区、清洁流程区和核心物近义变体。
- 第四池回填改成“时间物 + 环境设备 + 毯卷地垫 + 小型盆栽 + 图册画册”结构，尽量让它更像宠物角前台自然可见的场景补充物，而不是宠物用品整理区。
- 后续复看时又发现 `xx卷` 和 `xx画册` 仍然偏密，因此继续按“每种后缀最多保留两个”的口径微调，把 `针织毯卷 / 格纹毯卷 / 防水毯卷 / 绒毯卷 / 午睡毯卷 / 条纹地毯卷 / 爪印地毯卷 / 吸水垫卷 / 防滑垫卷 / 森林画册 / 海边画册 / 城市画册` 替换为 `条纹毯面 / 格纹毯面 / 防水席面 / 猫窝组合 / 食盆组合 / 玩具组合 / 脚印毯面 / 骨头毯面 / 格纹席面 / 爪印席面 / 拼色毯面 / 磨爪组合`，把第四池从“卷物和读物串联”再拉回一点到宠物角软装物、用品组合和前台地面物的混合结构。
- 本轮又继续压缩了 `毯面 / 席面` 家族，把 `条纹毯面 / 格纹毯面 / 脚印毯面 / 拼色毯面 / 格纹席面` 改写为 `牵引阵列 / 项圈阵列 / 磨牙阵列 / 外出分区 / 洗护分区`，避免第四池再次被同后缀软装词接管读感。

## 8. 修改后复审

- 当前四池数量均为 50，且池内唯一。
- 第四池与其余三池无重叠。
- 第四池本轮本地复查未命中禁词，且全部通过 `_is_medium_scene_fragment_item` 的中型场景边界校验。
- 这说明本轮已经先把“结构性错误”修掉了，尤其是第四池不再被 `盒 / 包 / 架 / 筒 / 桶 / 托盘 / 清洗盆` 这类家族主导。
- 从读感上看，第二池已经明显回到工具池；第四池也从“宠物备品间”转回“宠物角前台补充物”。
- 进一步按后缀统计复查后，第四池里的 `卷 / 画册` 已分别压到 2 条，满足“每种后缀最多保留两个”的补充要求。
- 本轮继续复查后，第四池里的 `毯面 / 席面` 也已分别压到 `2 / 2`，当前仅保留 `骨头毯面 / 爪印毯面` 和 `防水席面 / 爪印席面`。
- 本节结论仅针对 `宠物+日常` 的定点复查，不代表全仓所有类别已全量通过。

## 9. 剩余风险与暂留理由

- 第一池里仍保留了少量容器壳和外出壳，例如 `粮食密封桶`、`零食罐`、`尿垫托盘`、`外出背包`、`便携航空箱`；这轮暂不动，是为了先稳住主体池，不把问题从第四池转移到第一池。
- 第三池仍保留了一些旅行洗护件和外出配件单位，例如 `一包香波旅行瓶`、`一盒指套牙刷`、`一套外出折叠碗`、`一盒胸背挂扣`、`一袋雨衣扣带`；它们当前格式合规，但后续如果继续做第三池精修，可以再轻压一次。
- 第四池虽然已经脱离“收纳壳链”，但现在仍有 `钟 / 历 / 机 / 卷 / 册` 这些自然家族；后续若继续微调，重点应放在压模板密度，而不是重新引入 `盒 / 包 / 架 / 筒 / 盆`。
- 这轮已经把 `卷 / 画册 / 毯面 / 席面` 压到较合理密度；后续若继续微调，优先关注 `图册 / 摄影册 / 写真册` 这类相邻读物家族，而不是重新把卷物或软垫面词补回来。
- 当前版本已经可以作为后续同类修改的合格基线，下一步若继续优化，建议优先做“第一池容器壳轻压”和“第三池旅行配件轻压”，而不是再大改第二池和第四池。
