# 全局重复词定位清单

用途：记录跨类别反复出现、已影响整体多样性的高频词或词族，供后续统一修改；本文件不替代 `审查文件/{类别} 审查文档.md`。

## 记录规则

- 只记录已经明显影响读感、复用过密或正在扩散的词 / 词族。
- 位置统一写法：`类别 / 池名 / 条目 / blind_boxes.py:行号`。
- 若问题是词族，标题写词族名，位置里写具体条目。
- `统一修改方向` 只写整改方向，不直接在这里写替换候选词。
- 状态只用：`待定位`、`待修改`、`已修改`、`已复查`。
