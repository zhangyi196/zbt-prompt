# 表情库 2.0.1 目录草案

用途：为 `组图 23 表情库.md` 的后续重构提供可审目录。当前文件只确定顶层表情类别、分类结构、核心面部表现和禁止表现；暂不写每个表情的 8 个具体表情变体，也不写单人 / 多人模板。

版本说明：
- `v2.0.1` 将原 30 + 30 类收敛为正向 24 类、负向 24 类。
- 合并或替换实际脸型近似的类别，例如 `生气 / 恼羞成怒` 合并为 `生气`，`奸笑` 替换为 `吐舌调皮`，`嘴硬坚定` 替换为 `咬牙硬撑`。
- 顶层表情名优先使用能直接画出脸部状态的名称；心理判断、剧情结果和人物策略下沉到适用场景或具体变体。
- 后续落库时，每个顶层表情再扩展为 8 个具体表情变体，并分别提供单人 / 多人模板。

## 一、通用写法边界

| 项 | 内容 |
|---|---|
| 可写部位 | 眉、眼、脸颊、额头、嘴、牙齿、舌头、口边、脸侧 |
| 可用卡通锚点 | 爱心瞳孔、星星瞳孔、旋涡眼、豆豆眼、小鸭嘴、鼓脸、红晕、黑线、青筋、短烟线、小汗、冷蓝 / 冷灰 / 冷青额头 |
| 额头规则 | 额头只写局部卡通情绪标记，如小汗、青筋、黑线、短烟线、冷蓝 / 冷灰 / 冷青小色块；不得写整脸变色、真实光影、发光或大面积阴影 |
| 口边规则 | 口水、吐气线、呕吐短线只停在嘴边；不得喷溅到衣服、背景或画外 |
| 禁止内容 | 身体动作、头部转向、四肢动作、发型、衣服、饰品、背景、真实伤口、真实呕吐物、文字云 |

## 二、正向表情目录（24 类）

| 分类 | 表情 |
|---|---|
| 漂亮心动 | 喜欢、崇拜、感动、害羞 |
| 明快开心 | 开心、满足、自豪、松眉微笑 |
| 期待兴趣 | 馋嘴、期待、惊喜、好奇 |
| 俏皮可爱 | 卖萌、假乖、甜美撒娇、顽皮 |
| 坏笑捣蛋 | 得意、忍笑、吐舌调皮、眯眼坏笑 |
| 藏情绪反差 | 看戏、冒汗假笑、憋笑嘴硬、偷乐憋笑 |

### 正向表现边界

| 表情 | 要表现的内容 | 禁止表现 |
|---|---|---|
| 喜欢 | 爱心瞳孔、脸颊红晕、甜笑、小露齿 | 不写痴呆脸、夸张口水、身体靠近 |
| 崇拜 | 星星眼、亮晶晶圆眼、嘴微张赞叹 | 不写跪拜、举手、身体仰头 |
| 感动 | 泪光、柔眉、嘴角发颤但上扬 | 不写大哭、拥抱、感谢动作 |
| 害羞 | 红脸、偷看、抿嘴笑、眼神躲开 | 不写低头、捂脸、扭身体 |
| 开心 | 弯眼、大笑、整齐露齿、脸颊鼓起 | 不写跳起、挥手、全身兴奋 |
| 满足 | 半闭舒服眼、软笑、脸颊圆润 | 不写睡着、瘫软、摸肚子 |
| 自豪 | 自信亮眼、眉尾上扬、嘴角稳定翘起 | 不写挺胸、叉腰、抬下巴 |
| 松眉微笑 | 眉眼放松、轻微笑、嘴角柔和回升 | 不写疲惫脸、叹气云、身体松垮 |
| 馋嘴 | 盯目标、舌尖、小口水、期待嘴 | 不写真实进食、伸手拿物、口水喷溅 |
| 期待 | 大亮眼、眉尾抬起、嘴微张 | 不写惊恐 O 嘴、身体前扑 |
| 惊喜 | 瞳孔放大、眉毛上扬、开心小张嘴 | 不写恐惧、尖叫、后仰动作 |
| 好奇 | 歪眉、圆眼、嘴角微歪、疑问感 | 不写转头探身、问号文字云 |
| 卖萌 | 圆眼、小鸭嘴、鼓脸、软眉 | 不写动物耳朵、外来器官、幼化身体 |
| 假乖 | 圆眼装无辜、抿嘴、单侧眉藏笑 | 不写光环、双手装乖、道具 |
| 甜美撒娇 | 红晕、软眼、嘟嘴、甜笑 | 不写身体摇晃、拉扯他人、过度媚态 |
| 顽皮 | 眯眼、舌尖、小尖牙、单边坏笑 | 不写手势、脸部真实变形 |
| 得意 | 单边挑眉、嘴角翘高、眼神发亮 | 不写胜利手势、叉腰、身体炫耀 |
| 忍笑 | 抿嘴压笑、弯眼、脸颊憋起 | 不写爆笑、捂嘴动作 |
| 吐舌调皮 | 舌尖露出、圆眼或眯眼、单边眉、小坏笑 | 不写长舌头、鬼脸过度、身体动作 |
| 眯眼坏笑 | 半眯眼、单侧嘴角上扬、露小牙 | 不写心理词“幸灾乐祸”、指人动作 |
| 看戏 | 半眯眼、嘴角轻翘、旁观冷笑 | 不写爆笑、转头看远处、夸张兴奋 |
| 冒汗假笑 | 额头小汗、眼神飘、嘴角硬拉 | 不写“装没事”剧情过程、身体僵住 |
| 憋笑嘴硬 | 嘴巴压住笑、脸颊鼓起、眼睛藏笑 | 不写争辩台词、明显大笑 |
| 偷乐憋笑 | 嘴角偷偷上扬、眼睛发亮、笑意收住 | 不写庆祝动作、大笑、胜利姿态 |

## 三、负向表情目录（24 类）

| 分类 | 表情 |
|---|---|
| 愤怒对抗 | 生气、不耐烦、咬牙硬撑、懊恼 |
| 判断警觉 | 困惑、怀疑、警惕、害怕 |
| 紧张暴露 | 心虚、慌张、焦急、僵住 |
| 尴尬石化 | 尴尬、社死、无语、石化 |
| 委屈吃瘪 | 委屈、吃瘪、破防、失落 |
| 低落排斥 | 哭、笑着哭、反胃、犯困睡着 |

### 负向表现边界

| 表情 | 要表现的内容 | 禁止表现 |
|---|---|---|
| 生气 | 压眉、怒视、红脸、咬牙、额头青筋 / 短烟线 | 不写挥拳、跺脚、全身发抖 |
| 不耐烦 | 半眯眼、眉头压低、嘴角下撇、额角小青筋 | 不写摆手、转身离开 |
| 咬牙硬撑 | 牙齿咬紧、嘴角绷住、眼神不服、额头小汗或短青筋 | 不写大喊、哭、嘴角塌掉、握拳 |
| 懊恼 | 皱眉、嘴角绷住、额头黑线 / 短烟线 | 不写抓头、拍脑袋、后悔身体动作 |
| 困惑 | 皱眉、豆豆眼、嘴微歪 | 不写问号文字云、挠头动作 |
| 怀疑 | 单边挑眉、斜眼、嘴角收紧 | 不写指认、靠近查看 |
| 警惕 | 瞳孔缩小、眉峰收紧、嘴闭紧、小汗 | 不写防御姿势、后退 |
| 害怕 | 瞪大眼、冷蓝额头、小汗、牙齿打颤 | 不写逃跑、抱头、全身颤抖 |
| 心虚 | 眼神躲闪、冷蓝额头、额头小汗、僵硬假笑 | 不写“被抓包”事件词、后退、肩膀缩起 |
| 慌张 | 眼神乱、眉毛高低错位、嘴发紧、额头汗点 | 不写手忙脚乱、肩膀变化 |
| 焦急 | 眉心上拱、嘴角紧张、额头汗滴 | 不写奔跑、催促手势 |
| 僵住 | 眼睛定格、嘴巴卡住、冷灰额头 | 不写身体姿态冻结 |
| 尴尬 | 黑线、红脸、僵笑、眼神躲避 | 不写社交剧情解释、身体缩小 |
| 社死 | 空白眼、冷灰额头、嘴僵成细线 | 不写倒地、灵魂出窍、背景文字 |
| 无语 | 半垂眼、嘴成直线、额头黑线 | 不写叹字云、夸张翻白眼 |
| 石化 | 冷灰额头、空白眼、石化短线、嘴僵住 | 不写真实裂纹、身体变石头 |
| 委屈 | 小鸭嘴瘪嘴、泪光、眉尾下垂 | 不写跪求、抱住别人 |
| 吃瘪 | 鼓脸、不服眼、嘴巴憋住 | 不写受伤、被打、真实肿胀 |
| 破防 | 嘴角塌掉、泪花、红脸或黑线 | 不写撕扯衣服、崩溃身体动作 |
| 失落 | 下垂眉眼、嘴角下落、脸颊失去弧度 | 不写低头、坐倒、背景阴影 |
| 哭 | 眼泪、皱眉、张嘴哭脸 | 不写泪水喷到衣服或背景 |
| 笑着哭 | 眼里泪花、嘴巴还在笑、眉尾下垂 | 不写“强颜欢笑”抽象心理、身体强撑 |
| 反胃 | 冷青额头、舌头歪出、嘴边呕吐短线 | 不写真实呕吐物、污染衣服、喷溅 |
| 犯困睡着 | 黑眼圈、眼袋、半闭眼、嘴巴微张、睡意失焦 | 不写趴下、低头、睡姿变化；睡眠 Z 只能在脸侧外留白 |

## 四、后续落库要求

- 每个顶层表情固定补 8 个具体表情变体。
- 8 个变体必须有不同主视觉锚点，避免只做同义换词。
- 每个变体后续再扩写 1 条单人模板和 1 条多人模板。
- 模板行使用 `编号. 具体表情变体；字段：……；字段：……` 格式。
- 每条模板写 3-5 个脸部字段，至少包含 `眼：` 或 `嘴：` 之一，每个字段最多出现 1 次。
