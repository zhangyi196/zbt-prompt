# 表情库 2.0.0 版本草案

用途：为 `组图 23 表情库.md` 的 2.0.0 重构提供可审目录。当前文件只确定顶层表情类别与每类 8 个具体表情变体，暂不写单人 / 多人模板。

核心口径：
- 顶层类别保留简洁情绪词，便于前置提示词选择。
- 具体表情变体负责美式卡通趣味，承载脸颊、额头、舌头、牙齿、口边短线、脸侧汗滴等可画锚点。
- 每类固定 8 个变体；同一类内不得重复，同一极性内避免大面积近义堆叠。
- 后续落库时，每个变体再扩写为 1 条单人模板 + 1 条多人模板。
